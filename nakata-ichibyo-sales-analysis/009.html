## 良い点

**引き出せた深い情報:**
- 「離婚も真剣に考えた」
- 「自己肯定感が下がっている」
- 「更年期で行動がついていかない」
- 「現実逃避でスマホを見てしまう」
- 「夫に2回『出ていけ』と言われた」
- 「私が自分を大切にしてあげないと私がかわいそう」

### 2. 担当者のメンタル体験による深い共感（◎）

担当者自身がブラック部署で追い詰められた経験を語った。

**自己開示の内容:**
- 「食料品の買い物ができなくなった」
- 「何も買わないで帰ってくることが何度も続いた」
- 「判断ができなくなる。決められなくなる」
- 「IQが下がったと思った」
- 「このままだと死んじゃうかもしれないと思った」

**効果:**
- 顧客が「分かってもらえている」と感じた
- 「能力じゃない」「環境のせい」という言葉の説得力が増した

### 3. 強い承認の繰り返し（◎）

自己肯定感が低下している顧客に対し、何度も強い承認を伝えた。

**使用した承認フレーズ:**

- 「能力じゃないですよ」

- 「ご自身を絶対責めないでください」
- 「自己肯定感が戻ってくれば本来のご自身が戻る」

### 4. 「自分を大切にする」テーマの一貫性（◎）

セッション全体を通じて「自分を大切にする」というメッセージを伝え続けた。

**伝えた言葉:**
- 「自分を守れるのは自分だけ」

## 改善点

### 1. 環境確認の不足（重大）

開始時に「この後のご予定」「周囲にご家族は?」の確認がなかった。

**結果:**
- 夫が何度も入室し、担当者の説明が中断
- 子供（親戚の子）が5回以上入室
- 最終的に「ご飯だから」と呼び出されて終了
- 2時間30分以上のセッションになり、契約手続きができなかった

**対策:**
開始時に必ず確認する:
```
「今日は大体60分〜90分ほどお時間いただく予定ですが、
この後のご予定は大丈夫ですか?

また、今いらっしゃる場所は、周りにご家族とかいらっしゃいますか?
もし途中で呼ばれそうでしたら、先にお伝えいただければ
中断しても大丈夫ですので。」
```

### 2. クロージングで引きすぎた（マニュアル失敗パターン3）

担当者が「私からあの、進めたりっていうことはしないです」と言った。

**問題:**
- 顧客は「じゃあ考えます」「明日にします」と言いやすくなる
- 背中を押す機会を逃した

**言うべきだったトーク:**
```
「なお子さん、さっき『私が自分を大切にしてあげないと私がかわいそう』
っておっしゃいましたよね。

この講座に参加するって、まさに『自分を大切にする』行動だと思うんです。

背中を押させてください。」
```

### 3. 「やりましょうよ！」を言っていない

顧客が「参加したいな」と言った時点で、さわやかに「やりましょうよ！」と言うべきだった。

### 4. SPIN営業法（損失金額）がない

顧客は「毎日探し物をしている」「買って満足」と言っているが、金額化していない。

**聞くべきだった質問:**
- 「探し物に1日何分くらい使っていますか?」
- 「買って満足、使わないまま置いてある物、いくらくらいありそうですか?」

### 5. アンカリングがない

価格提示前に、他の高額投資や損失金額との比較がない。

**聞くべきだった質問:**
- 「韓国語のレッスンはいくらですか?」
- 「マシンピラティスのパーソナルはいくらでしたか?」

### 6. 時間管理の問題

2時間30分以上のセッションになり、最後は駆け足になった。
契約手続きまでたどり着けず、翌日に持ち越し。

## 新規パターン（マニュアル更新候補）

### 成功パターン候補: 自分のメンタル危機体験を語る

担当者自身が追い詰められた経験を具体的に語ることで、「この人は分かってくれる」という深い信頼を構築する。

**事例（高見直子さま 2026年4月）:**
- 顧客が「やる気が出ない」「行動がついていかない」「更年期かも」と語った
- 担当者が自身のブラック部署での経験を語った:
  - 「食料品の買い物ができなくなった」
  - 「何も買わないで帰ってくることが何度も続いた」
  - 「判断ができなくなる」「IQが下がったと思った」
  - 「このままだと死んじゃうかもしれないと思った」
- 顧客は「私もまさにそう」と共感し、心を開いた
- 「能力じゃない」「環境のせい」という言葉の説得力が増した

**ポイント:**
- 抽象的な「大変でしたね」ではなく、具体的な体験を語る
- 「私も同じだった」というメッセージで安心感を与える
- そこから回復した経験を語ることで「変われる」という希望を与える

### 失敗パターン候補: 環境確認の不足で中断多発

開始時に環境確認をしなかったため、セッション中に何度も中断が発生し、クロージングまでたどり着けなかった。

**事例（高見直子さま 2026年4月）:**
- 開始時に「この後のご予定」「周囲にご家族」を確認しなかった
- 夫が3回以上入室し、説明が中断
- 子供（親戚の子）が5回以上入室
- イヤホンのバッテリーも切れた
- 最終的に「ご飯だから」と呼び出されて終了
- 契約手続きができず、翌日に持ち越し

**教訓:**
- 開始時の環境確認は「あったら良い」ではなく「必須」
- 特に海外在住・同居家族がいる顧客は注意が必要
- 中断が多いと顧客の集中力も途切れ、クロージングが難しくなる

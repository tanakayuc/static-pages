この資料について

チャレンジローンチを構築するための全工程を、ステップバイステップで進められるように整理したシートです。「今、自分が全体のどこを作っているか」を確認しながら、1ステップずつ進めてください。

★ まずはこれを理解する

田中祐一の「最新AIの使い分け」

プロモーション制作では、1つのAIで全部やろうとせず、3つのAIをそれぞれの得意分野で連携させるのが田中流。 この3点セットを揃えるところからスタートします。

1

Cursor

ライター

起点。プロモ素材をMDで整える

  • 企画・構成・コピーライティング
  • ターゲット設定・訴求軸の整理
  • 文章や企画案をドキュメント化
  • 必要に応じて素材の方向性も整理
2

ChatGPT

デザイナー

Cursorの内容をビジュアル化

  • Cursorで作った内容をもとにビジュアル・デザイン案を生成
  • 画像・レイアウト・図解などで「伝わる形」に可視化
  • 構成やデザインの意図を整理し、Claude Codeへ引き渡す
3

Claude Code

コーディング＆エンジニア

Claude Codeで「カタチにして届ける」

  • デザイン指示・要件をもとに実装
  • コーディング／ディレクション／エンジニアリングを一括推進
  • クオリティの高いプロモーションをスピーディに完成
Cursorで「伝える内容」を設計 → ChatGPTで「伝わる形」にデザイン → Claude Codeで「カタチにして届ける」
準備編：これを揃えてから始める

事前に揃える「ツール」

🔧 必要ツール（インストール／アカウント作成）

  1. CursorLPライティング・画像指示書の作成に使う
  2. ChatGPT（GPT-4o）画像生成に使う。無料でもOKだがPlus推奨
  3. Claude CodeHTMLコード生成・サーバーへの公開に使う

LP制作に必要な「素材」と「公開先」

📦 LP制作前に用意する素材（4点）

  1. コンフィグデータ案件の基礎情報（コンセプト・ターゲット・オファー等）→ コンフィグフォルダ
  2. 挨拶動画の台本工程4 Step23で作ったDay0台本
  3. 参考LP構成業種・トーンが近いLPを1〜3本 → オプトインLP素材フォルダ
  4. わしづかみコンセプト工程2で作ったプロモコンセプト・キャッチコピー

🌐 公開先の選択（3つから1つ選んでおく）

完成したHTMLをインターネット上に公開するための「置き場所」です。LP制作を始める前に、どれを使うか決めてアカウント準備だけしておきましょう。

  1. UTAGE + MCP連携既にUTAGE契約がある人向け。Claude Codeから直接アップできる（MCP接続の初期設定が必要）
  2. GitHub Pages無料。GitHubアカウントを作るだけで使える。初心者におすすめ
  3. Vercel無料枠あり。GitHubと連携して自動デプロイ。複数ページ管理に向いている

本番運用

📋 運用フェーズ プロモ期間中

素材が揃ったら、いよいよ本番運用。集客→ライブ→限定性→販売→クロージングを期間内に走らせる。法定書面も忘れずに。

56

集客の実施

人間
INPUT
セットアップ投稿・広告・サンキューページ・教育グループ
操作
SNS投稿／企画ライブ／個別声かけ／広告／号外の組合せで集客
OUTPUT
目標オプト数の達成
注意
セットアップ期間（約2週間）が結果を9割決める
57

ライブの実施

人間
INPUT
Day1〜Day5の台本
操作
ライブ実施／課題アップ／録画アップ／質問回答／課題フィードバック
OUTPUT
5日間のライブ完了
注意
録画は当日中にアップ。質問は翌日までに回答
58

限定性のプッシュ（1ステップ販売）

人間
INPUT
販売ページ・限定特典
操作

OUTPUT
販売期間の完了
注意
最終日の駆け込みが3〜5割を占める

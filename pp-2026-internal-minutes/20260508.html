1. 浅野悟：プロデューサーのAI活用基準とSNS自動化

前提：AI活用の「基準」はあるのか

浅野さんから「田中さんが前の作業会でプロデューサーとしてAIをこのぐらい使いこなすべき、という基準を話していたか」という質問から始まった。田中は「そんな基準はない」と即答。

田中が語ったのは、自分自身が最近、外注を一切使わずプロモーションを自力でやっている体験談だった。「時給で言ったら下がってる。今は練習中だから」という前置きをしつつ、ランニングページからセールスレターまで自分で作ってみている、という状況を共有した。

Cursor（ライター）とClaude Code（デザイナー/コーダー）の役割分担

浅野さんが「Cursorで自動化してるということ？」と聞いたところで、田中が自分の中での役割分担を整理して説明した。

  • Cursor = ライター担当：ライティングをCursorに担当させる。本気の自分案件はClaude Opus 4.7、制作受け案件はClaude 4.5と使い分けている
  • Claude Code = デザイナー/コーダー担当：レイアウト・デザイン・コーディングはClaude Code側に渡す。宴（Utage）へのアップロード指示や、Lora Artを使った画像生成なども Claude Code から連携して指示を出している

「業務の自動化」と「制作」は別の話だという整理も田中から出た。自動化の例として挙がったのは、セールスチャットの毎日自動ダウンロード、YouTube書き起こしの自動化、蓄積データへの週1フィードバックとマニュアル整備の自動実行などがある。

ミニアプリを作って動かすという考え方の説明では、「YouTubeのURLを取得して書き起こしするアプリを一度パソコン内に作ると、それを呼び出すだけで動くようになる」という流れを田中が説明した。アメブロの記事を全部抜き出す処理も同様で、AIが「このロジックでアプリを作っていいですか？」と確認してくるので「作って」と言えばミニアプリができるという。

フルAIコンテンツへの懸念と現実的なSNS戦略

浅野さんはさらに「オーガニックのインスタ投稿をAIに任せて自動化するのはどう？」と聞いた。田中の答えは「AIで作った投稿は面白くない」というシンプルなものだった。

  • YouTubeではフルAIで作ったコンテンツの収益剥奪が話題になっており、再生が回らなくなっている
  • 「人間Aが書いた台本を人間が読み上げる」のはOK。フルAIは今後ますます優遇されなくなる方向
  • インスタも同様の流れが来る可能性がある

田中が「次チャレンジしたい」として提案したSNS戦略が以下：

  • コンテンツホルダーに週1でライブをさせる
  • そのライブをAIで切り取って編集し、ショート動画としてアカウントに上げる
  • AIが考えたタイトル・テーマを事前に用意しておいて「これについて話してください」とライブで回答してもらう
  • その切り抜き編集を全AI化する

「人間のコンテンツをAIが編集する」という順序がポイントで、「フルAIコンテンツを作る」のとは逆だという説明だった。

浅野さんが「確認ですが、人間がやったものをAIにサポートさせるということ？」と整理すると、田中は「そう。ヒューマンパワーでやるものをAIにサポートさせる」と答えた。

広告とインスタの関係については「広告見た人がインスタを調べることは明らかにある。広告アカウントとインスタは相互でぐるぐる回る」として、信頼できるコンテンツをインスタに置いておくことが生存戦略だと田中は言った。ただし「まだSNS側まで手が回ってない、自分でも週1ライブを義務付けてスタッフにフルAIでインスタ運用させようかと思っている段階」という正直なコメントも。

最後に田中が言ったのが「AIには夢見ない方がいい。でもできないと生産性の差はすごい」という一言で、浅野さんとのやりとりを締めた。

2. 進藤雄介：ペットタクシー事業の集客課題

PPC広告で問い合わせゼロ

進藤さんが抱えている課題は、ペットタクシー事業での集客が全く動いていないこと。ペット探偵事業は広告を出せば集客できていたが、ペットタクシーは広島と大阪でPPC広告を出しても問い合わせがほぼゼロ。47都道府県に広げることを検討中だという相談だった。

田中が即座に返したのは「東京以外に需要ないんじゃない？みんな車持ってるじゃないですか」という話。進藤さんは「高齢で免許返納した方や若い免許ゼロ世代もいる」と補足したが、田中は「そもそもペットタクシーという言葉を知らない人が大多数」という本質的な課題を指摘した。

「検索広告はある程度認知が高まって初めて成立するもの。ペット探偵は困ったときに検索するが、ペットタクシーは検索ワードにたどり着かない」という分析だった。

地域コミュニティ形成という提案

田中が提案したのは、直接クロージングに繋げるアプローチではなく、コミュニティと地域特化インスタを使ったUGC（口コミ）戦略だった。

  • 「大阪ペット図鑑」のような地域×ペット特化のインスタアカウントを作る
  • アカウントでは大阪のおすすめ動物病院、大型犬が泊まれるホテル、ペット可の公園といった情報を発信する
  • そこに集まってくるペット好きのユーザーがコミュニティを形成し、その中でペットタクシーの利用者が口コミを投稿していく
  • UGCが積み重なることで「ペットタクシー」という言葉の検索ニーズが自然と生まれていく

田中が参考として挙げたのが「東京デート図鑑」というインスタを運用してマッチングアプリでマネタイズしていた事例。地域ごとの恋愛系アカウントをたくさん作っていた20代経営者で、「地域別専用インスタの勧め」というノートを過去に出していたと記憶している、という話だった。

「グルメや不動産の地域特化アカウントはあるけど、ペット×地域ってまだない可能性が高い。大阪でおすすめの動物病院5選、みたいな投稿は大阪の人が食いつくアカウントになるんじゃないか」という田中の仮説。

進藤さんも「確かに、入り口をペットタクシーに絞らなければ引きも強い。ホテルも探偵もタクシーも総合的にやってる会社として口コミが積み上がっていったら強い」と整理し、「ペットタクシーのペの字もないけど中でサービス案内が入る」というアカウント設計イメージを掴んでいった。

田中は最後に「大阪ペット図鑑、そのままいけそう」と具体的なアカウント名案まで出してこのセクションを締めた。

案件①：女性向けダイエット/魅力アップ案件

キャッチコピーのテスト予算について

北山さんから「広告テストにいくら必要か」という相談も出た。300〜500万円規模でしかテストしたことがないが、クライアントが小予算しか出せない場合どうするか、という問いだった。

田中が整理したのは「何をテストするかによって予算は全然変わる」という考え方。

  • キャッチコピーのクリック率（CTR/CPA）だけを確かめるなら、1パターン1万円 × 10パターン = 10万円で十分
  • LPのヘッダー画像と広告クリエイティブを全く同じにして、広告セットを分けて10パターン出す
  • 10パターンの中から上位3つに絞れる
  • 「制約するかどうか」のテストはまた別で、そちらはもっと予算が必要だが、それは職人気質に任せるしかない部分もある

北山さんは「じゃあまずキャッチコピーのクリック率から見る、10万円テスト」で動くことを整理した。

原田コーチ

✅強み・スタンス

正解の人生設計やローンチ型への押し付けを避け、受講生の言葉（「楽しく生きたい」「目標がない」など）を否定せず強みとして再定義する。

停滞・参加不足を責めず、すでに動いている事実（SNS、外部案件、AIシェア、イベント準備）を拾い上げてから次の一歩を提案する。

大きなコミットより「一回だけ」「気軽に」「2〜3人でも」のハードルを下げ、約束で区切る場面が多い。

よく使う打ち手

  1. PLCを義務ではなく「環境の使い方」（ローン地塾の比喩）として説明し、参加プレッシャーを外す
  2. 勉強会・ワークショップ・雑談チャット・リアル接点で他受講生の工夫を拾う提案
  3. 売上・タスクだけでなく「何のために」「理想の未来から見ると今何をするか」で感情とモチベーションを戻す問い
  4. イベント・セミナー前の「じわじわ」セットアップ（ティザー、種まき、PP募集の流れ参照）
  5. プロデューサー提携で「止まらない仕組み」を作る話（週1だけでは責任が及ばない構造の指摘）

向いている受講生像

モチベーションや方向性の停滞、PLC参加が薄い、時間の使い方に悩む、自己売り込みが苦手、複数の入り口案を抱えて整理したい、というタイプ。

⚠️注意（コーチング設計上）

次回日程を必ず押さえる場面と、本会話内で未設定のまま終わる場面がある。フォローは事務・チャット側の確認が必要なことがある。

大脇コーチ

✅強み・スタンス

受講生主導の持ち込みを長く受け止め、要所で枠組み（価格帯、コミュニティ二種類、最初の一手）を一段上げて整理する。

成果がある受講生では、祝賀と「なぜ動けたか」の言語化を厚くし、PLCの設計思想（コミュニティ・テンプレ・比喩）に接続する。

マーケ実務（LINE、個別フォロー、セールススライド、広告タイミング）に寄せた具体提案が得意。抽象論のあと「誰に・いつ・何を聞くか」で着地する。

よく使う打ち手

  1. 「そうですね／なるほど」で同調したうえで、いま一番最初にできる一手に戻す
  2. 高タッチとスケールのジレンマには、合宿一括以外の運営オプションを並べて提示
  3. グルコン・田中さん個別・認定講師・録画FBなど、相談先を名前付きで振り分ける
  4. プロデューサー候補には「Webができるか」より実行・提案の具体性・報酬・Plan B を先に固める質問
  5. 未達を責めず、なぜできなかったかを対策（スケジュール・余白）に変換する

向いている受講生像

ローンチ・チャレンジ経験者のスケール設計、副島型のマーケ・商品・LINEリスクの整理、大平型のセールス泥臭い改善、高野型のプロデューサー・外注判断。

⚠️注意

追加コーチング枠と通常PLCコーチング回数が別枠の受講生がいる（大平和幸など）。ファイル名の回数を正とする。

石田コーチ

✅強み・スタンス

対人のもやもやを「伝達が未熟」「まだ伝わり切っていない」に還元し、AIシミュレーション・ミーティング書き起こし・電話という具体ツールで次の一手を渡す。

プロデューサー案件・サンチャレ観察を「大量インプットの課題」として具体化し、他講師比較・参加者心理・施策の細部までメモする運用を繰り返し示す。

コーチング終了時（救済・追加枠）に、プロチャット・認定講師個別・追加コーチングの相談導線を整理して渡す。

よく使う打ち手

  1. チャット偏重の擦り合わせを、短時間の通話・Zoomに寄せる
  2. グルコンは対立や設計の軸、田中さんは「これでいいか」より「こう進めようと思うがFBが欲しい」という出し方を勧める
  3. AIはツール触りより、業務1件に紐づけた改善サイクル（下書きAIとレビューAIの分離など）
  4. 竹見さん・浦野さん型の三者関係では「まず理解と行動」の順序に落とし込む
  5. LP・Canvaは「型があれば入れ替えで済む」と足場をかけたうえで、発信・人間関係へ広げる

向いている受講生像

チーム内の認識差・プロデューサーとの温度感、サンチャレを深く学びたいプロデューサー志望、伝達と添削プロセスの期待値すり合わせ。

⚠️注意

救済コーチングは2回で終了する運用の説明が入る（大澤など）。以降は認定講師個別コンサル等へ誘導。

吉岡コーチ

✅強み・スタンス

加入直後・3チャレ直前など、実行摩擦（ログイン、チャット、メンバーサイト、事務局連絡）の解消に時間を割く比重が大きい。

雑談・近況を単なるアイスブレイクで終わらせず、体力・見た目・行動継続を実行戦略の土台に接続する。

期限と移動予定（渡航・講座）を前提に逆算し、「今回は何を捨てて何を残すか」を早く決める。

よく使う打ち手

  1. 45日ロードマップと専用チャットで、いま完了させるタスクを画面共有しながら確認
  2. マーケの中身は田中さん・グルコン、コーチングは段取り伴走、という役割分担の明確化
  3. 広告・ハウス・LP・動画を同時に触る場面で、目的別に役割を分ける（例: Metaはテスト、本番母集団はハウス）
  4. 外注前に世界観・訴求軸を自分で固める重要性の共有
  5. 次回報告できる行動単位まで落として締める（日程・窓口の再確認）

向いている受講生像

高本型の蓄電池・高単価BtoB、外注と自走の板挟み、3チャレ逆算と制作オペレーションの整理。

⚠️注意

回数表記とセッション冒頭の「初めて」等が食い違うことがある。ファイル名の回数を運用正とする（高本2回目の注参照）。

第1部: 受講生タイプ（現時点の分類）

タイプは重なりあり。1名が複数に該当する場合がある。該当例はナレッジ登録時点のファイル名（回数・コーチ）で示す。

タイプA: 停滞・再参加・ハードル下げ型

特徴: 「進んでいない」「熱が入らない」「自己売り込みが苦手」「PLCにあまり出ていない」と語りやすい。コーチは欠如ではなくフェーズとして受け止め、小さなアウトプット・一回だけの約束・コミュニティの覗き見へ橋渡しする。

該当の例: 草薙1（原田）、嶺島1（原田）。華崎1は時間不足だが参加再開の文脈あり。

コーチングでよく出る論点: PLC活用、ワークショップ・雑談、別ルートでの成功の許容、物販への回帰など。

タイプB: 初動・オンボーディング・段取り伴走型

特徴: 加入直後またはロードマップ直後。ログイン、チャット、メンバーサイト、事務局連絡、45日タスクの完了が主戦場。ビジネス全体は広く聞きつつ、いまの45日で何を終わらせるかに引き戻す。

該当の例: 高本2（吉岡）。

コーチングでよく出る論点: 役割分担（田中＝マーケ中身、コーチ＝段取り）、プロデューサー探索の順序、数値・時間軸のすり合わせ。

タイプC: ローンチ経験後・スケール設計型

特徴: チャレンジ・オプト・成約など成果があり、なぜ動けたかの言語化から、講座設計・キャパ・月次チャレンジ・卒業後コミュニティまで視野が広がる。

該当の例: 高野1（大脇）。

コーチングでよく出る論点: ハウス・SNS・広告の順序、リストの質、ターゲット表現、専門家（渡辺さん等）へのリファラル。

タイプD: プロデューサー・外注・判断停滞型

特徴: 候補・外注・PLC式が並立し、判断が止まりやすい。基準・契約・Plan B・忙しさの理由の言語化が中心。

該当の例: 高野4（大脇）。

コーチングでよく出る論点: 成果報酬、社内Web担当の共有範囲、Kindle＋既存リスト、PLC告知ルール、XMindと動画視聴。

タイプE: チーム内コミュニケーション・伝達型

特徴: プロデューサー・コンテンツホルダーとの「伝わっていない」感、チャット疲れ、下書き感覚のギャップ。電話・書き起こし・AI反復で打開。

該当の例: 大澤2（石田）。

コーチングでよく出る論点: 訴求と商品論理の一貫性、グルコン・田中への出し方、認定講師個別コンサルへの移行。

タイプF: プロデューサー案件・サンチャレ深掘り型

特徴: ワークショップ後の案件獲得、竹見・浦野型の三者関係、サンチャレを運営・参加者両視点で大量観察。AIは業務1件に紐づけた改善サイクル。

該当の例: 平尾3（石田）、鴇田3（石田）。

コーチングでよく出る論点: 温度感の確認、他講師サンチャレ比較、イベントでの自己紹介、個別コーチング終了後の相談導線。

⚠️タイプG: 実務セールス・個別フォロー改善型

特徴: LINE制約、個別相談の温度感、セールススライドの組み立て、チャレンジ〜セールスのスケジュールが具体。

該当の例: 大平和3（大脇）。

コーチングでよく出る論点: ヒアリングで相手に危機感の言葉を言わせる設計、テンプレの商材別即決、グルコン・認定講師への振り分け。

タイプH: 大型イベント・事前育成型

特徴: セミナー・説明会のクロージング、じわじわセットアップ、コンセプト未確定のまま期限が近いイベントを先行。

該当の例: 山本大2（原田）。

コーチングでよく出る論点: プロデューサーで強制力、AI叩き台、コンセプトの入口変更（AI活用寄りのテストマーケ）。

タイプI: 商品転換・時間・モチベ再点火型

特徴: 旧商品縮小・新商品準備、SNSとハウスのミスマッチ、やるべきことは多いがワクワクが消えやすい。「何のために」の問いで再接続。

該当の例: 華崎1（原田）。

コーチングでよく出る論点: リアル入り口の多角化、月末売上目標と今週の動画・ステップメール、YouTuber像との接続。

タイプJ: 持ち込み多・優先順位・コミュニティ設計型

特徴: グルコン示唆を土台に、ライブ・QR・Kindle・コミュニティ二層・LINEリスクまで横に広がる。最後に「最初の一手」に戻す。

該当の例: 副島1（大脇）。

コーチングでよく出る論点: 価格帯別販売、合宿代替、次に誰にどこで聞くか。

タイプK: 3チャレ逆算・集客導線・制作オペ型

特徴: 添削・LP・広告・ツール課金で混乱しつつ行動力は高い。渡航など制約下で捨てる／残すを決め、ハウス再活用・広告はテスト寄り。

該当の例: 高本3（吉岡）。

コーチングでよく出る論点: 短間隔3チャレ、MetaフックAB、コンテンツ量産の型、外注前の世界観固め。

LPヘッドデザイン指示書

LPヘッドデザイン指示書

目的

LPのヘッド部分で「これは自分のための話だ」と思わせ、LINEオープンチャット登録またはセールスページの精読へ進ませる。

この指示書は単体ではなく、参考サンプル とセットでChatGPTやCodeXへ渡す前提です。参考サンプルによって、余白、文字量、画像方向、CTA位置、スマホ表示の解釈を揃えます。

オプトインLPヘッド

採用メッセージ

地味で平凡な会社員向け
才能・経験・顔出し不要の
裏方起業のロードマップを公開

サブメッセージ

社長の右腕として活躍する、プロモーション戦略を担う希少価値の高い「裏方Webマーケターの道」。

視覚方向

  • キラキラした起業家感ではなく、静かでプロフェッショナルな戦略感
  • 主人公は「表に立つスター」ではなく「売上を支える裏方」
  • ノートPC、ファネル図、数字、チャット、設計図のような要素を入れる
  • 会社員でも入りやすい清潔感と、売上に近い仕事の緊張感を両立する

避ける表現

  • 派手な札束、過度な自由人表現
  • 顔出しインフルエンサー風
  • 汎用的なオフィス写真だけの構成
  • 「楽して稼げる」印象

画像生成/デザイン指示

```
地味で平凡な会社員が、表に立つ起業家ではなく、社長の右腕としてプロモーション戦略を支える世界観。ノートPC、ファネル図、売上グラフ、チャット画面、設計資料が並ぶ静かな作戦室。落ち着いた白とチャコール、信頼感のあるグリーン、アクセントに淡いゴールド。人物は背中や横顔程度で、顔出し不要の裏方感を出す。スマホでも文字が読みやすい余白を確保。
```

セールスページヘッド

採用メッセージ

「実績がない」「顔出しは苦手」
その真面目さが、あなたの可能性を止めているとしたら？

商品名

45日間WEBマーケター超実践ブートキャンプ

サブメッセージ

知識を増やすだけではなく、社長の右腕として売上に関わる最初の実践経験を作る45日間。

視覚方向

  • チャレンジ後に「次の一歩」が見えている状態を表現
  • 5日間の学習から、45日間の実践環境へ進む導線を見せる
  • 個人の孤独な挑戦ではなく、チームや実践の場がある安心感を出す
  • 期限と限定性は強く、ただし煽りすぎない

Claude/HTML制作向け指示

  • ファーストビュー内に商品名、対象者、変化の約束、CTA、締切を入れる
  • CTAは「詳細を見る」ではなく、参加意思が前に進む文言にする
  • スマホでは縦長で、CTAが1画面目下部に見えるようにする
  • ヘッド画像は文字を焼き込みすぎず、HTML側で可変テキストとして重ねる
  • LPの下層は「旧世界/新世界」「真の原因」「解決策」「実践環境」「特典」「締切」の順で展開する
参考サンプル

LPヘッド制作時に、デザインの方向性を合わせるための参考情報を置くファイルです。

今回の参考方向

  • セールスレポートのように、読みやすく余白を確保する
  • 文字は詰め込みすぎず、ファーストビューで約束と対象者が分かるようにする
  • 画像は派手な起業家風ではなく、裏方Webマーケターの戦略感を出す
  • CTAはスマホでも最初の画面内で見える位置に置く

参考として渡すもの

  • 気に入っているLPやレポートのスクリーンショット
  • 近いトーンのファーストビュー
  • 避けたいデザイン例

差し替えメモ

実案件では、このファイルに参考URL、スクリーンショット、デザインの好き嫌いを追加してから、LPヘッドデザイン指示書 と一緒にChatGPTやCodeXへ渡します。

LPヘッドデザイン指示書の使い方

このフォルダは、LPのファーストビューを制作するための素材セットです。

ChatGPTやCodeXに依頼するときは、以下をセットで渡します。

  • LPヘッドデザイン指示書
  • 参考サンプル
  • オプトインLP原稿

依頼時の前提

指示書だけでは、完成イメージがぶれやすくなります。参考サンプルも一緒に渡し、コピー、余白、文字量、画像方向、CTA位置、スマホ表示のイメージを揃えます。

依頼文の例

```
以下のLPヘッドデザイン指示書と参考サンプルをもとに、オプトインLPのファーストビュー案を作ってください。
目的は、登録前に「自分のための話だ」と感じてもらい、VSL視聴とメール登録へ進ませることです。
スマホ表示を優先し、コピー、画像方向、CTA、余白の設計まで提案してください。
```

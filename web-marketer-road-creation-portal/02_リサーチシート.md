# リサーチシート

## リサーチ目的

「WEBマーケターへの道」が市場の中でどのポジションを取るべきかを整理し、LP、動画台本、配信、セールスページに使う訴求の軸を決める。

## 市場カテゴリ

- 副業、起業、フリーランス化
- Webマーケティングスクール
- SNS運用、広告運用、ライティング、デザインなどの単体スキル講座
- コンテンツ販売、プロダクトローンチ、オンライン講座販売
- AI副業、AIコンサル、デジタル人材育成

## 見込み客が比較する可能性のある選択肢

| 選択肢 | 魅力 | 取りこぼし |
| --- | --- | --- |
| SNS起業/インフルエンサー型 | 自分の名前で売れる、華やか | 顔出しや発信が苦手な人には重い |
| Webデザイン/制作スクール | スキルが明確で始めやすい | 時給型、作業者型に留まりやすい |
| 広告運用スクール | 数字に近く収益化しやすい | 全体設計やオファーには踏み込まないことが多い |
| ライティング講座 | LPや文章に強くなる | 導線全体や販売後まで見えにくい |
| 高額ビジネス講座 | 売上に直結しやすい | 初心者には心理的ハードルが高い |
| AI副業系 | 新しさと話題性がある | 実践環境や実績作りが曖昧になりやすい |

## ライバルが強く訴求しがちなこと

- 月収、自由、独立、在宅ワーク
- SNSで発信して個人ブランドを作る
- 広告運用や制作などの単体スキルを学ぶ
- AIを使って効率よく稼ぐ
- 高単価案件を獲得する

## ライバルが弱くなりやすいこと

- 顔出しや自己主張が苦手な人への導線
- 初実績を安全に作る実践環境
- 売上に直結する全体設計の理解
- 社長の右腕としての立ち位置
- チームで成果を作る経験
- 会社員気質、裏方気質を強みに変える言語化

## 狙う空きポジション

「スター型起業」でも「単体スキル習得」でもなく、起業家のプロモーションを裏方から支え、売上に関わる実績を作るWebマーケター。

## ずらし方

| 軸 | ずらし方 |
| --- | --- |
| 性別 | 性別よりも、顔出しや自己主張への抵抗感を軸にする |
| 年齢 | 若いSNS強者より、社会人経験があり真面目に支えられる層を狙う |
| 世界観 | キラキラした自由ではなく、地味でも売上に貢献する実務感を出す |
| ノウハウ | 単体スキルではなく、DECODE/ファネル全体設計を武器にする |

## ターゲットの感情

- 目立つ人だけが勝つ世界に疲れている
- 自分の真面目さや裏方気質は、ビジネスでは武器にならないと思っている
- スキル学習だけでは将来が変わらないと薄々感じている
- 「売上を作る現場」に入ってみたいが、入り口がわからない
- 一人で案件を取るのは怖い

## 訴求アイデア

1. 顔出し不要で、売上に近い裏方の仕事を目指せる
2. 自分の商品がなくても、起業家のプロモーション支援で実績を作れる
3. スキル単体ではなく、売れる仕組み全体を理解する
4. 45日間の実践環境で、最初の成果に近づく
5. 地味で平凡な会社員の真面目さを、社長の右腕として活かす

## LP/動画に使うべき論点

- なぜ会社員こそWebマーケターを目指すべきなのか
- なぜ顔出しや実績がなくても始められるのか
- なぜ単体スキルではなく、全体設計が重要なのか
- なぜ最初の成果には実践環境が必要なのか
- なぜ裏方Webマーケターは今後価値が高いのか

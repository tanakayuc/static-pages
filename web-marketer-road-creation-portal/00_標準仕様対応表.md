# 標準仕様対応表

このファイルは、WEBマーケターへの道の制作パッケージが、共通の制作パッケージ標準仕様をどこまで満たしているかを示すものです。

参照標準: `../../_制作パッケージ標準テンプレート/00_制作パッケージ標準仕様.md`

## 完了条件チェック

| 条件 | 対応状況 | 根拠 |
| --- | --- | --- |
| コンセプト、リサーチ、オファー、工程表、制作物一覧がMarkdownで存在する | OK | `01_`〜`05_` の各ファイル |
| 工程表がファネルモデルに合っている | OK | 直販型チャレンジローンチとして整理 |
| 制作物一覧に原本URL、状態、次アクションがある | 一部OK | 原本URLと状態はあり。次アクションは追加管理が必要 |
| LP/セールスページのヘッド指示がある | OK | `06_ヘッドデザイン指示書.md` |
| 配信管理が分類されている | OK | `07_LINE配信管理シート.md` |
| HTMLポータルで全体を辿れる | OK | `index.html` |
| 未確定事項が分離されている | OK | `08_レビュー確認事項.md` |
| 原本差し替え時の更新対象がわかる | 一部OK | 制作物一覧に管理項目あり。自動再生成ルールは今後実装 |

## この案件で採用したファネル

| 項目 | 内容 |
| --- | --- |
| ファネル | WEB直接販売型チャレンジローンチ |
| 導線 | オプトインLP -> LINEオープンチャット -> 5日間ライブ/課題 -> セールスページ -> 決済 |
| 申込導線 | 個別相談なし |
| KPI | オプトインLP成約率、チャレンジ有効参加率、セールスページ遷移率、購入率 |

## 他案件へ転用する時に差し替える項目

| 差し替え項目 | 内容 |
| --- | --- |
| ファネルモデル | 個別相談、体験セミナー、セミナーセールス、4本動画など |
| KPI | ファネルごとに申込率/参加率/成約率を変更 |
| 配信チャネル | LINE、メール、Chatwork、Discord、Facebookグループなど |
| 制作物一覧 | ページ、台本、スライド、動画、配信、特典を案件に合わせる |
| ヘッド指示 | LP種別とターゲットに合わせる |
| 公開範囲 | 社内原本、クライアント確認用、一般公開用を分ける |

## 残すべき改善

- 実競合名の追加
- 実績値ベースのKPI更新
- 価格/支払い方法/保証の正式化
- Day5ライブシナリオURL、サンキューページURLの確定
- HTMLポータルの自動生成ルール化

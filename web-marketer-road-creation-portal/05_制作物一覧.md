# 制作物一覧

この一覧は、田中祐一AIが新規制作モードで作る素材を整理するためのものです。

制作の主導線は工程表です。工程表の番号に沿って作り、作った素材は以下の3階層に分類します。

## 1. 集客フェーズの素材

| 制作物 | 役割 | 納品形 |
| --- | --- | --- |
| SNSセットアップ指示書 | SNS起点の案件で、プロフィール、固定投稿、導線、投稿テーマを決める | 指示書 |
| 広告クリエイティブ指示書 | 広告画像、広告文、遷移先、テストパターンを決める | 指示書 |
| オプトインLP原稿 | 登録前に誰へ何を約束するかを伝える | 原稿 |
| LPヘッド指示書 | ファーストビュー、画像方向、CTA、スマホ表示を指示する | デザイン指示書 |
| オプト前VSL台本 | LP上で登録前の教育と選別を行う | 台本 |
| オプト後VSL台本 | 登録後にOC参加とDay1着席の期待値を高める | 台本 |
| VSLスライド指示書 | VSLにスライドが必要な場合、画面化する内容を指示する | スライド構成案 |
| サンキューページ原稿 | 登録直後に正式参加へ進める | 原稿 |
| オプトイン自動返信メール | メール登録で止まった人を正式参加へ戻す | 配信原稿 |

## 2. 価値提供フェーズの素材

| 制作物 | 役割 | 納品形 |
| --- | --- | --- |
| LINEオープンチャット全体ポータル | 参加者が最初に見る全体案内を作る | 投稿原稿 |
| 固定投稿 | ライブ案内、課題、特典、参加ルールを常設する | 投稿原稿 |
| 通常配信 | Day別のリマインド、課題、アーカイブ、Q&Aを送る | 配信原稿 |
| Day1〜Day5ライブ台本 | 各日の導入、本編、課題、次回予告を作る | 台本 |
| Day別スライド指示書 | ライブにスライドが必要な場合、1日ごとの構成案を作る | スライド構成案 |
| 課題フォーム構成 | 各日の課題提出内容と提出後導線を決める | 構成案 |
| 提出特典/コンプリート特典案内 | 課題提出と完走を促す案内を作る | 原稿 |

## 3. 販売ページの素材

| 制作物 | 役割 | 納品形 |
| --- | --- | --- |
| 販売前メッセージ原稿 | Day5後に販売導線へ進む理由を作る | 配信原稿 |
| セールスレター原稿 | 問題提起、商品内容、価格、特典、保証、申込導線を作る | 原稿 |
| セールスページヘッド指示書 | ファーストビュー、CTA、締切、証拠の見せ方を指示する | デザイン指示書 |
| 販売期メール原稿 | 販売開始、理由、事例、不安解消、締切を送る | 配信原稿 |
| 販売期LINE原稿 | 公式LINEでレター案内、質問回答、締切、終了案内を送る | 配信原稿 |
| 個別説明会ページ原稿 | 説明会型・個別相談型ファネルの場合に申込ページを作る | 原稿 |
| 購入完了ページ原稿 | 決済後の案内、参加導線、次アクションを伝える | 原稿 |

## 管理方針

| 管理項目 | 理由 |
| --- | --- |
| 工程番号 | 工程表のどこで作る素材かを追えるようにする |
| フェーズ分類 | 集客フェーズの素材、価値提供フェーズの素材、販売ページの素材のどこに属するかを分ける |
| 納品形 | 原稿、台本、構成案、デザイン指示書、スライド構成案を区別する |
| 実装対象 | UTAGE、LINE、メール、スライド、HTMLなど、後工程で入る場所を示す |
| 添削観点 | 完成後に何を見直すかを残す |

## 注意

実装後のページURLは、田中祐一AIが作る制作物そのものではありません。必要な場合は、納品後の実機反映・添削モード側で管理します。

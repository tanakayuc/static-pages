# オファーシート

## 商品概要

| 項目 | 内容 |
| --- | --- |
| 商品名 | 45日間WEBマーケター超実践ブートキャンプ |
| 商品カテゴリ | 実践型Webマーケター育成プログラム |
| 販売方式 | セールスページから直接販売 |
| サポート期間 | 45日間 |
| 個別販売導線 | なし |
| 価格 | サンプル仮値: 60,000円。正式価格は要確認 |
| 支払い方法 | サンプル仮値: 一括払いを基本。分割/決済条件は要確認 |
| 購入完了ページ | 後工程で実装先へ反映。ここでは原稿を作る |

## 商品コンセプト

地味で平凡な会社員が、顔出しや強い実績に頼らず、社長の右腕としてプロモーションを支えるWebマーケターになるための45日間実践環境。

## 変化の約束

自分の商品や大きな実績がなくても、プロモーション全体の構造を理解し、売上に関わる実践経験を積むことで、Webマーケターとしての最初の実績を作る方向へ進める。

## なぜその変化が可能なのか

- DECODEを軸に、集客から販売までの全体構造を学ぶ
- ライブと課題で理解を深める
- 実践環境の中で、売上に近い活動に関わる
- 一人で案件獲得を目指すのではなく、チームや場の力を使う
- 田中祐一のプロモーション実務の視点を学べる

## 提供内容

| 項目 | 内容 |

| 環境 | 実践コミュニティ/チーム型の学習環境 |
| サポート | サンプル仮値: グループ支援、課題確認、質問対応。個別サポートの範囲は要確認 |
| 成果物 | 実践記録、提案材料、Webマーケターとしての活動設計 |

## 特典

| 種別 | 内容 |
| --- | --- |
| 課題特典 | Day1〜Day5の課題提出特典 |
| オファーシート | Day3特典として用意 |
| コンプリート特典 | 初心者のためのwebで売れる型、地味な起業で食っていく、Webマーケターマインドセット |
| PLC導線 | 一定条件でPLC参加時に差額参加できる案内あり。条件は原本確認 |

## 購入障壁と解消策

| 購入障壁 | 解消策 |
| --- | --- |
| 実績がない | 最初の実績を作るための実践環境として打ち出す |
| 顔出しが苦手 | 裏方Webマーケターという立ち位置を提示する |
| スキルが足りない | 単体スキルより全体設計を学ぶと伝える |
| 自分に案件が取れるか不安 | チーム、場、実践環境を強調する |
| 価格への不安 | 将来の成果報酬、上流支援、右腕ポジションの価値で納得感を作る |

## 緊急性/限定性

- 販売締切を設定する
- 実践環境の参加枠を限定する
- チャレンジ参加直後の熱量が高いタイミングで案内する
- 販売終了後は募集終了メッセージを出す

## 実績/仮KPIメモ

| 項目 | 内容 |
| --- | --- |
| 参加実績 | 販売終了後ログに「ブートキャンプには30名の方にご参加」の記載あり |
| 仮単価 | 60,000円 |
| 仮売上 | 30名 x 60,000円 = 1,800,000円 |
| 注意 | 上記はサンプル仮値。正式な価格、実売上、返金/分割状況は別途原本で確認する |

## セールスメッセージの骨子

1. 知識だけでは変わらない
2. 最初の実績を一人で作るのは難しい
3. 必要なのは、売上に関われる実践環境
4. 地味で真面目な人ほど、裏方Webマーケターに向いている
5. 45日間で、社長の右腕として動ける視点と経験を作る

## 要確認事項

- 正式価格
- 分割払いの有無
- サポート体制の詳細
- グループコンサル/個別フィードバックの範囲
- 返金保証の有無
- PLC差額参加の正式条件

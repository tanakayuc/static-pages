# WEBマーケターへの道 制作ポータル

このフォルダは、「WEBマーケターへの道」プロモーションの制作物を整理した、田中祐一様用の制作ポータルです。

目的は、今後の案件で「ログインした相手ごとに、どのような制作物が揃うのか」を見せるための基準を作ることです。細かな作業手順ではなく、各フェーズで何を決め、どの成果物が完成するかに絞っています。

## 収録ファイル

| ファイル | 役割 |
| --- | --- |
| `00_案件マニフェスト.md` | 案件の基本情報、原本リンク、成果物チェック |
| `00_標準仕様対応表.md` | 共通標準仕様に対するこの案件の対応状況 |
| `01_コンセプトシート.md` | ターゲット、ポジション、旧世界/新世界、コアコンセプト |
| `02_リサーチシート.md` | 市場、ライバル、空きポジション、訴求の切り口 |
| `03_オファーシート.md` | 本命商品、約束、提供内容、特典、緊急性 |
| `04_工程表.md` | ファネル全体の工程と、各工程で完成するアウトプット |
| `05_制作物一覧.md` | 実際に必要になるページ、配信、台本、素材の一覧 |
| `06_ヘッドデザイン指示書.md` | オプトインLPとセールスLPのヘッド制作指示 |
| `07_LINE配信管理シート.md` | 事務連絡、定期配信、Q&Aを分けた運用整理 |
| `08_レビュー確認事項.md` | レビューで出た論点と田中祐一への確認質問 |
| `index.html` | 上記を一つのポータルとして閲覧するHTMLサンプル |

## このサンプルのファネル

動画を埋め込んだオプトインLPからLINEオープンチャットに登録し、5日間のライブと課題で価値提供します。Day5でレターを受け取りたい方に公式LINE登録を促し、公式LINE内で期間限定のセールスレターを公開するモデルです。個別相談を挟まない直販型として整理しています。

```
オプトインLP
  -> サンキューページ
  -> LINEオープンチャット
  -> 5日間ライブ / 課題
  -> Day5で公式LINE登録
  -> 公式LINE内で期間限定セールスレター公開
  -> 決済 / 購入完了ページ
```

## 共通テンプレート

他案件へ転用する時は、次の標準テンプレートを複製して使います。

- `../../_制作パッケージ標準テンプレート/README.md`
- `../../_制作パッケージ標準テンプレート/00_制作パッケージ標準仕様.md`
- `../../_制作パッケージ標準テンプレート/templates/`

## 使い方

1. `index.html` を開く
2. HTML内の「工程表」「制作物一覧」「コンセプトシート」「リサーチシート」「オファーシート」を見る
3. 必要に応じて各Markdownを編集する
4. 原本を差し替える場合は、対応するMarkdownを更新し、`index.html` を再生成する
5. 公開用にする場合は、HTML化して `static-pages` 側に反映する

`index.html` が、この案件の制作ポータルです。工程表と制作物一覧は別URLに分けず、同じHTML内に入れます。画面左側のサイドバーで工程表、制作物一覧、主要シートへ移動し、右側の本体で各成果物をビジュアルに確認します。各Markdownは閲覧用ではなく、編集・差し替え・ダウンロード用の原本として扱います。

標準テンプレートは、別案件を新しく作るための空の型です。完成サンプルを確認する入口ではありません。

このフォルダは制作モードの原本です。公開用ページではなく、The Lead側の制作データとして扱います。

## 注意

価格、支払い方法、保証、サポート範囲など、原本から確定できない項目は「サンプル仮値」と「要確認」を分けています。公開サンプル化する前に、実案件の確定情報へ差し替えてください。

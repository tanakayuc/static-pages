# FV画像 広告用差し替え指示書

作成日: 2026-05-11
対象: たかみず保江「次世代型AIセールス」オプトインLP 広告用

## なぜ差し替えが必要か

Meta広告の審査でアカウントが停止された。FV画像内のテキストをMeta AIで審査リスク判定した結果、**「成約率8割へ」が高リスク**と判定された。画像はテキスト焼き込み済みのため、画像自体の修正が必要。

## Meta AIによるリスク判定結果

| リスク | テキスト | 判定理由 | 対応 |
|--------|---------|---------|------|
| **高** | **成約率8割へ** | 具体的な成果・数値の断定。非現実的な結果の約束 | **変更必須** |
| 中 | わずか10分で | 時間・労力の断定。ビジネス系は厳しめ | 変更推奨 |
| 中 | "売れる人"に | Before/Afterの示唆。個人の状態を断定 | 変更推奨 |
| 低 | 読み上げるだけで | 簡便性の強調だが単体では比較的通る | そのまま |
| 低 | AI共鳴覚醒シナリオ | 造語なのでほぼ問題なし | そのまま |
| 低 | 次世代型セールス | 低リスク | そのまま |
| 低 | 期間限定で募集します | 緊急性だが基本OK | そのまま |

## 変更箇所（最小限）

現在の画像テキスト:
```
わずか10分で
あなたを"売れる人"に

AI共鳴覚醒シナリオを
読み上げるだけで

成約率8割へ

次世代型セールス
無料勉強会

期間限定で募集します
```

変更後:
```
わずか10分で                    ← 中リスク: 判断に任せる
あなたを"売れる人"を目指す        ← 中リスク:「に」→「を目指す」に変更推奨

AI共鳴覚醒シナリオを             ← そのまま
読み上げるだけで                 ← そのまま

成約率アップを目指す             ← 【必須変更】「8割へ」→「アップを目指す」

次世代型セールス                 ← そのまま
無料勉強会                      ← そのまま

期間限定で募集します             ← そのまま
```

**最低限やること: 「成約率8割へ」→「成約率アップを目指す」の1箇所だけ。**

中リスクの2箇所も変えるなら:
- 「わずか10分で」→「短時間で学べる」
- 「あなたを"売れる人"に」→「あなたを"売れる人"を目指す」

## デザイン指示

- **レイアウト・構図・人物写真・背景は一切変更なし**
- テキスト変更は上記の1〜3箇所のみ
- フォントサイズ・色・装飾は現行を完全踏襲
- 「成約率アップを目指す」は現在の「成約率8割へ」と同じ位置・ゴールド大文字

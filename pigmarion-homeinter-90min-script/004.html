画像生成指示書：対象者セクション（こんな方にオススメ／ご遠慮）

説明会LPのセクション12「対象者」をHTMLから画像化するための生成指示書。
オススメ7項目／NG3項目を信号機型のコントラスト構成で1枚に収め、ターゲットを瞬時に判別できる視覚演出にする。

1. 完成イメージ

縦長1枚絵（縦910〜1100px × 横600〜750px相当・スマホ縦長想定）。
上半分（明るい・ティール系）に「オススメ7項目」、下半分（落ち着いた・コーラル系）に「ご遠慮3項目」。
中央で背景色がふわっと切り替わる構成。

2. ファイル仕様

項目 値
ファイル名 target_audience.png（生成）→ target_audience.webp / target_audience.jpg（最適化後）
画像比率 縦長 約 4:5〜2:3（モバイル縦長LP想定）
推奨解像度（生成時） 1200×1500〜1800px
配置先 説明会ページ/images/
HTMLでの参照サイズ 約 1200×1600px相当

3. デザイン仕様

全体トーン

  • 既存LPと同じ世界観：清潔・知的・温かみ
  • 背景は白〜パステル中心
  • 装飾要素は抑えめ（FV画像と並べたときのリズムを崩さない）

カラーパレット（厳守）

用途 カラー
メイン強調（オススメ側） #00B4CC ティールブルー
サブ強調 #008B9C ディープティール
背景上部（オススメ側） #E6F7F9 ライトティール
背景下部（NG側） #FFF0F0 薄いコーラル
NG側のアクセント #FF6B6B コーラル
本文テキスト #333333
補足テキスト #666666
ホワイト #FFFFFF

タイポグラフィ

  • 見出し：太めゴシック（Noto Sans JP Bold相当）
  • 本文：通常ゴシック（Noto Sans JP Regular〜Medium）
  • 重要語のハイライトは黄色マーカー風のうっすらした下線
  • 見出しは「ホームインターメソッド」のロゴ感に合わせ、明朝の混在も可（ただし軽め）

4. レイアウト構成

上段：こんな方にオススメです！（信号機の青）

[小さなアイコン群：本・地球儀・親子のシルエット] ─ 5%幅・装飾控えめ
　
セクションヘッダー（中央寄せ・大きく）
こんな方に
【オススメ】 です！

☑ 毎日「早くしなさい」と言い続けてイライラしている方

☑ ワンオペ育児で孤独を感じている方

☑ 「自分で考えて動ける子」に育てたいと思っている方

下端の余白

下端は20px程度の余白を残し、HTML側の「次のCTA」セクションへの誘導を作る。

5. 装飾アイコン（軽量化のため最小限）

  • 上段の見出し横：☀️ or ✨ 系の小さなアイコン（ティール）
  • 下段の見出し横：🚫 or ⚠️ 系の小さなアイコン（コーラル）
  • 7項目／3項目の左端アイコンは✓と×のみ（追加アイコン不要）

6. NG指示

  • 派手なグラデーションは使わない（エステ系・スピ系の派手な雰囲気を避ける）
  • 顔写真は入れない（受講生・講師写真とは別レイヤー）
  • アニメーション・GIF不要（静止画のみ）
  • 文字に強い影付けは禁止（読みやすさ優先）

7. 軽量化（生成後の処理）

# WebP（最大幅 1200px）
python3 - << 'PY'
from PIL import Image
img = Image.open("target_audience.png").convert("RGB")
w,h = img.size
if w > 1200:
    nh = int(h * 1200 / w)
    img = img.resize((1200, nh), Image.LANCZOS)
img.save("target_audience.webp", "WebP", quality=82, method=6)
img.save("target_audience.jpg", "JPEG", quality=80, optimize=True, progressive=True)
PY

目標サイズ:
- WebP 100KB以下
- JPEG 200KB以下

8. HTML差し替え後の構成

完成後、index.html の対象者セクション（id="target"）を以下に置き換える:

<section class="bg-light" id="target">
  <div class="container">
    <picture class="img-block">
      <source srcset="images/target_audience.webp" type="image/webp">
      <img src="images/target_audience.jpg" alt="こんな方にオススメです（オススメ7項目／ご遠慮3項目）" width="1200" height="1600" loading="lazy" decoding="async">
    </picture>
  </div>
</section>

既存のHTMLチェックリスト2種類（オススメ／NG）と関連する .ng-list CSS のうち、本セクション以外で使っていないものは削除して軽量化する。

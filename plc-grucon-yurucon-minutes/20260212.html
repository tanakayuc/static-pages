1. もりたにさん：LPファーストビュー＆ダウンセル戦略

相談内容①

  • LPの1行目の訴求文をどうするか
  • 「一過性の変容で終わらない」vs「話して変わる支援から形に残る支援へシフト」

フィードバック

ファーストビューの問題点
- 両方とも「何言ってるかわからない」
- 一瞬で理解できないコピーはNG
- 「定着」という言葉は説明しないとわからない

改善案
1. 顧客の言葉をそのまま使う
- 「前のセッションでは変わったんですけど、また落ち込んでしまいました」がなくなる新しい心理技術
- 吹き出しでも可

  1. 専門用語を避ける
  2. 「心理技術」「定着」→ 読みづらくしている可能性

  3. カウンセラーって書いてるならカウンセリングはわざわざ言わなくてOK

  4. 要素を減らす

  5. 「涙される、深く、優しい、癒し」→ 4要素は多い、せめて3つ

コンセプトの方向性
- 「定着」より「他人軸→自分軸」の方が伝わる
- 会話以外のアプローチ（絵本） が興味性高いならストレートに出す
- 1,3,4番はそのままでOK、2番を見直す

相談内容②

  • 3Daysワークショップ（3〜4万円）をダウンセルとして売ったが引き上がらない

フィードバック

高額商品後のダウンセルの真実
- 第一オファーで買わなかった人へのダウンセルは、本命と合わない人が買っている
- 短期的には引き上がらない（1年後、2年後、3年後の顧客作り）
- 「情報が欲しいだけの人」「その場で問題解決できればいい人」

逆パターンなら引き上がる
- 初手で3万円のワークショップ → バックエンドという流れなら可能性あり
- 最初から安い商品を買った人は、高額商品を買う素地がある

ダウンセルの最適設計
- 3日間のワークショップはダウンセルとしては重い
- ワンデーセミナー＋1回カウンセリング のパック → 関係性強化
- グループで体験 → 「1対1でちゃんと見てほしい」と思わせる

ハウスリスト向けの施策
- 9,800円〜3,000円のワークショップ体験
- グループカウンセリング付き
- そこから個別相談に誘導

2. 伏見美映さん：セールスチーム構築の相談

相談内容

  • プロデュースしている人がセールスに自信がない
  • 商品を売ったことがない人にセールスチームをつけるべきか

フィードバック

経験不足 vs 売れるけど売らないタイプの見極め

タイプ 対応
売れるけど忙しくて売れない セールスチームつける
単純に経験不足 「まず10件やってから文句言え」

丸投げマインドへの警告
- 「自分の案件なんだから自分でやれよ」が基本
- 課題感もフィードバックもないまま丸投げ → 成長しない
- セールスは乗り越えるべきポイント

プロデューサーの役割
- プロデューサーが録画を見る、一緒に参加する
- ああでもない、こうでもないとフィードバックする
- 全チームやっている

バランスの判断
- すべての案件に均等にエネルギーを注ぐのは不可能
- ここは突き放す、ここはがっつり入る → それぞれ判断
- 外部活用（石田さんのセールス講座など）も選択肢

フィードバックのポイント
- 何が課題だったのか
- スライドのどこが説明しづらかったか
- 何が伝わりづらかったか
- → これだけでもデータが取れる

3. 木田由香利さん：フロントコミュニティ戦略

相談内容

  • 真子さんの日本の英知伝道師案件
  • 体験セミナー来た人に3万円のコミュニティを売り、3ヶ月後にバックエンドオファー
  • 前回9人中7人がバックエンドに入学

フィードバック

広告からコミュニティ販売の収支計算
- 体験セミナー誘導コスト：約4万円/件（うち全案件平均）
- 成約単価：約20万円（PLCのような遠い案件は40万円）
- 3万円のコミュニティ販売：1件売れるごとに-5万円

3万円コミュニティの問題点
- 100オプトインで3%購入 → 9万円の売り上げ
- 10万円コミュニティなら100人中2人で20万円
- 50万円商品なら100人中1人で50万円
- 同じ集客数で売り上げが5分の1〜6分の1になる

最適な価格設定
- 3万円はめっちゃ難しい
- 99,800円/年間 の方が広告カバーしやすい
- 3万円だからコンバージョン率が3分の1になるわけではない

LTV視点での計算
- 短期的な赤字が許容できるなら3万円で高速回転もあり
- 20人入って10人がバックエンド → 500万円以上
- ROASは長期で見ると400%も可能

改善案
1. 体験セミナーをやめて動画1本でコミュニティオファー
2. 口コミが起きやすい設計
3. アップセル（個別セッション+3万円）で平均単価を上げる
4. 審査制コミュニティ として運営ストレス軽減

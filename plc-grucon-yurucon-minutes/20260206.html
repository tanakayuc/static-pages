## 1. エニアグラムを活用したタイプ別マーケティング

### 概要
AIにエニアグラムのタイプを指定してコピーライティングを依頼することで、ターゲットに刺さるメッセージを効率的に作成できる。

### エニアグラムのタイプ別特徴

| タイプ | 特徴 | 刺さる訴求 |
|--------|------|-----------|
| タイプ2（助ける人） | 他者貢献重視 | 「感謝される」「必要とされる」「周りを幸せにする」 |
| タイプ4（成功者） | 実績で圧倒したい | 「ナンバーワン」「実績」 |
| タイプ5（知識欲求） | 詳細に学びたい | 「ノウハウ」「論理的説明」 |

### タイプ2への自己投資の口説き方
タイプ2は「自己投資＝悪」と思っている傾向がある。

**NG**: 「あなたが変わるために投資していい」
**OK**: 「あなたの自己投資は、周りの誰かを救うための投資につながっている」

### 活用のポイント
- ステップメールでタイプ別に訴求を変える
- 苦手なタイプは来ないようにする設計も有効
- 個別セールス前にヒアリングシートからタイプを想定

## 2. LPコンセプトの「真の原因」の作り方

### 「真の原因」の条件
1. 言われた瞬間に「え？何それ？」となる
2. 説明すればイメージがつく
3. 「このままだとヤバい」という感情を喚起する

### NG例とOK例

| NG | OK |
|----|----|
| アンバランス（普通すぎる） | オートバランス機能が壊れている |
| 慢性炎症（意味不明） | 自己修復力が枯渇している |
| バランスが悪い（抽象的） | 修復力と破壊力が逆転している |

### ポイント
- 顧客が認知していることは「当たり前」になる
- 認知していなくても、説明すれば納得できればOK
- 「なんとかせなあかん」と思わせる表現が必要

## 4. キンドル出版とセミナーの連動設計

### よくある失敗
本の販売とセミナーを同時に訴求すると、目的が混同してユーザーが混乱する。

### 成功パターン
```
1. キンドル無料キャンペーン
2. ランキング1位を取得
3. 「1位記念でセミナーやります」と発表
```

### 本とセミナーを分けるべき理由
> 「本のリリースを目的にセミナーに参加すると、セミナーの参加率がすごく下がる」

- 本が欲しいだけの層が離脱する
- 本を売ろうとするとバックエンドが売りづらくなる

### 連動させる場合の設計
```
LINE登録 → キンドル本が無料で受け取れる
         → さらにセミナーの案内も届く
         → 本をダウンロードした人はセミナー参加可能
```

## 5. 広告運用の最適化と数値改善

### 広告代理店の分散について

**結論：代理店を増やすと効率は下がる**

- 分散はリスクヘッジであって効率化ではない
- 同じクリエイティブで複数代理店を使うと「カニバリ」が起きる
- 数値を良くしたいなら一社に絞る

### 予算とCPAの関係
- 一気に予算を上げるとCPAが高騰する
- 予算を落として丁寧に運用すればCPAも下がる可能性

### 人数優先 vs 利益優先

| 目標 | 施策 |
|------|------|
| 人数を取りたい | 広告予算を維持、クールを増やす |
| 利益を確保したい | 広告を止めて無料セミナーで売る |

### 追いかけセミナーの活用
全クールを統合した追いかけセミナーを実施することで、広告なしで利益を積み上げられる。

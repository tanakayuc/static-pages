1. 年齢・遅いのでは

Q 年齢が高い・遅いのではと思っている

理想の回答
- 「自分も上の子は8歳超えてから入りました。全然大丈夫です」
- 「今からできるなら、今しかないですよね」
- 「下の子は7歳なので、まだ間に合います」
- 「あっという間に大きくなるから、1日でも早く」

補足 抽象的な安心ではなく、担当者自身の体験と数字で伝えると説得力が出る。

2. 教材・子供に合うか

Q うちの子に教材が合うか・合わなさそう

理想の回答
- 「自分は教材をがっつり3ヶ月やったわけではなく、ワークとフォローアップとサークルがメインで子供が変わりました」
- 「グローバルキッズマムに入られたら、ゆきさんもそこがメインになるんじゃないかなと思います」
- 「メインはママのOSを変えるワークとコミュニティで、教材は補助です」

補足 教材ありきの不安を外し、コミュニティ・ワーク中心の価値に誘導する。

3. 英語・診断のボコ

Q 診断で英語のところがボコ（低め）だった

理想の回答
- 「英語はご家庭ご家庭で考え方が違うので、ボコっていうかへこんでても大丈夫です」
- 「他言語活動をしているなら、土台はできてるってことです。やってない子よりストンと腑に落ちやすいと思います」

補足 不安を希望に結びつける。やれば伸びる、という方向に言葉を向ける。

4. 英会話・お家英語

Q 英会話に行きたくない・英語やめて・英語やだと言う

理想の回答
- 「行きたくない＝英語が嫌いとは限らないかもしれません。ママと一緒にいたいだけのこともあります」
- 「英語嫌いになる前に食い止めることが大事。英語はできるって思わせることが1番大事です」
- 「お家英語は遊びがメインで、英語はおまけ。英語やらされてる感をなくすんです」

Q ママ英語やめてって言われる／お家英語への自信がない

理想の回答
- 「英語が苦手な親御さんでも、動画教材のフレーズを繰り返すだけでいけます」
- 「子供の好きな遊び（ごっこ遊び、お医者さんごっこ等）と英語を組み合わせると食いつきが違います。教材にもありますよ」

Q DWE（ディズニー英語システム）を買ったが使わなかった。また同じことにならないか

理想の回答
- 「DWEと全然違います。駆け流しではなく、ママと一緒に遊ぶ中で英語に触れるスタイルです」
- 「動画を見せることが目的ではないんです。ママがポイントを理解して、日本語で遊ぶだけでも十分効果がある」
- 「1年間の期限があるので、ダラダラしない。DWEは一生使えるから逆にやらなくなった、という方は多いです」
- 「英語が苦手で駆け流しが苦痛だった方でも、工作動画やお料理動画なら子供が食いつくので、ママが無理に英語を頑張る必要はありません」

補足 DWEで100万使って使わなかった経験がある人は罪悪感を抱えている。「あなたが悪かったのではなく仕組みが違った」というフレーミングで過去を否定せず、今回は構造が根本的に違うことを伝える。「期限がある→やる」「日本語でもOK→苦痛がない」の2点が刺さる。（例：20260220成田やよいさん）

5. 旦那・家族の障壁

Q 旦那が頑張りすぎなくていいよ・普通でいいと言う

理想の回答
- 「お母さんが楽しそうになる→子供が楽しそうになる→お父さんも巻き込まれます」
- 「うちの旦那も最初はゲーム好きで、子供の前で携帯見たりYouTube見たりしないって私が言い出した時、自分も見れなくなるんだよって言ってました。でも私がやめたら子供もやめて、ご飯のあとブロックスやトランプで家族で遊ぶようになって、いつの間にかパパも参戦してました」
- 「えみ子先生と15分話す時間を設けられます。意思のある方対象です」

Q 今夜旦那にプレゼンする／相談してから決めたい

理想の回答
- 「旦那とメルマガを一緒に読んでいらっしゃるんですね。お互いの考えを共有し合う関係性が築けていて素晴らしいです」
- 「旦那さんがこう言いそうなときは、うちも最初そうだったけど子供が楽しそうになったら巻き込まれました、と伝えてもらえるとよいかもしれません」

Q 夫が教育に全く興味がない。プレゼンしても反対されそう

理想の回答
- 「もし認定講師講座を選ぶなら、お仕事としての可能性もお伝えいただくといいかもしれません。教育投資＋キャリアの両面だと経済合理性を重視されるタイプには響きやすいです」
- 「月々3万6,000円の分割なら、習い事1つ分と同じです。今の習い事と比較してどちらが子供のためになるか、という話にすると具体的になります」
- 「ご主人が教育に興味がないタイプであれば、金額と期間だけ簡潔に伝えて"あとはやらせて"がうまくいくケースもあります。旦那さんの性格に合わせてアプローチを変えましょう」

補足 教育に全く興味がない夫の場合、「教育内容」で説得するのは逆効果。「いくらかかるのか」「何が返ってくるのか」の2点に絞る方がよい。講師講座なら「収入を得られる可能性がある」が最強カード。過去の交渉成功体験（家・車・習い事など）がある場合はそれを引き出して「今回もできますよね」と促す。（例：20260220成田やよいさん）

Q 60万の講座を自腹で払った直後。夫へ小遣いの前借りを相談したい

理想の回答
- 「教育投資への理解が取れているなら、小遣いの前借りという資金調達方法は有効です。毎月まとめてお金をもらってる場合、早めにまとめて欲しいと言えばいい、とおっしゃってましたよね」
- 「認定講師講座なら収入の可能性もあるので、教育投資＋キャリアの2軸で説明しやすいです」
- 「今夜話して決まったらすぐ申し込めるように、法定書面の読み合わせは先に済ませておきましょう」

補足 直近の高額支出があっても、夫の教育投資への理解が取れていれば小遣いの前借りは現実的な選択肢。法定書面を先に読み合わせておくことで、相談後にすぐ申し込める状態にしておく。（例：20260226中江穂奈美さん）

Q 夫が「パッとやりたいはダメ」「説明してから決める」タイプ

理想の回答
- 「講座スケジュール、料金に何が含まれるか、月2500円で継続できること、を資料でお渡しします。口頭で説明するより、夫さんが自分で読んで判断する材料があると伝わりやすいです」
- 「えみ子先生と15分話す時間を設けられます。意思のある方対象です」

補足 費用対効果を重視し、論理的に判断する夫には、口頭の説得より具体的な資料を渡すのが有効。家具の購入などでも「なんで使うのか説明して」と求めるタイプは、自分で読んで納得する材料を欲している。（例：20260226中江穂奈美さん）

6. 再個別・前回諦めた

Q 前回は金額で諦めた／1年半前に聞いたがやめた

理想の回答
- 「どういう理由で、金額が高い印象だったんですか？」（まず理由を丁寧に聞く）
- 「今回は何が違いますか？ご主人との共通認識は前回と比べてどうですか？」（変わったことを一言掘る）
- 「前回と比べて、〇〇さんの中で変わったことありますか？」（聞きすぎず）

7. 時間・育休・復帰

Q 時間がない／育休中なのに時間が取れていない

理想の回答
- 「幼稚園に行ってる時間がお仕事。お家のことを学びの時間にできたら、関わる時間は変えずに質が上がるイメージです」
- 「復帰されても、子供と関わる時間は多分今とそんな変わらないと思います。その中で質を上げるメソッドです」
- 「帽子や服をクローゼットに戻す、お弁当箱自分で洗う、おにぎり握る…そういうところで楽になるので、復帰してから学びながら子育て楽にする、トリプル一石三鳥になります」

Q 時短復帰する／仕事と両立できるか

理想の回答
- 「講座は夜9時半とか10時とかから受けられます。子供を寝かしつけてからで大丈夫です」
- 「 Zoom の日程も、昼間と子供が寝てからの時間帯で選べます」

8. 診断・8つの能力

Q 診断で低いところが出た／でこぼこが気になる

理想の回答
- 「少ないからダメではなく、機会がなかっただけ。ちっちゃい子の場合は特にそうです」
- 「じゃあこれを教えればいいの？ではなく、本人が『これって開くとどうなるの？』『この形ってどうやったらできる？』と思える環境作りが大事です」
- 「丸くするのが目標。丸くなると、自分で目標を立てて計画する、人に声をかけて手伝ってもらう、手先の器用さで叶える…全部できるようになります」

Q 先回り・手を出しすぎている自覚がある

理想の回答
- 「やりすぎてた感はある＝気づいているってことです。そこから変えていけます」
- 「お茶碗の器の話、聞いてますか？8歳までにゆきさんなりの視点でやっていた。アンテナの高いママだったと思うので、しっかりやってらっしゃると思います」

Q 1日10分はできてる。でも伸びてない気がする

理想の回答
- 「10分はできてる、素晴らしいです。ただ、同じ10分でも、自分が苦手な分野（高知性・運動等）への働きかけが不足していることがあります」
- 「動画で『高知性』『運動能力』で検索すると、伸ばしたい分野に特化した動画がリストアップされます。今日はこれ、と選んで一緒に見てやってみる形です」
- 「1日10分、携帯を引き出しにしまって、お子さんだけを見る。その中身の質を変えていくイメージです」

補足 量はできているが、分野の偏りに気づかせ、「伸ばしたいところを具体的に知りたい」モードに転換する。（例：20260226岩崎明子さん）

9. 申込・クロージング

Q 今すぐ決められない／旦那に相談してから

理想の回答
- 「申し込んでも入金しなければキャンセルできます」
- 「金額キープで送っちゃうのもありだと思います」
- 「本日〇日以内のお申し込みでこの価格になります。まずは送って、相談してから入金する、でも大丈夫です」

Q 法定書面の読み合わせはいつ？

理想の回答
- 「今パパッとやるか、入金前にもう一度お時間取るか、どちらでも大丈夫です」
- 「一般的なものとほぼ同じです。変わってるところはほとんどありません」

10. 既存の教育投資との重複

Q 幼児教室とかぶる部分がある／今の教室を辞めるつもりはない

理想の回答
- 「辞める必要はまったくありません。むしろ〇〇教室でIQを伸ばしているなら、その土台の上にEQ・非認知能力を乗せるのが最も効率的です」
- 「トイズアカデミーさんは認知能力を伸ばすのが得意。うちは非認知能力とママのOSアップデートが得意。レイヤーが違うんです」
- 「Can do listやワークにかぶりを感じたかもしれませんが、うちのメソッドの本質はワークの中身よりも"ママの関わり方の設計"の部分。教室では教えてくれない部分です」

補足 教室を否定すると防御壁が上がる。「補完」の位置づけで、教室でやっていることを活かすフレームにする。IQが高い子ほどEQの重要性が増す（小学校での適応問題等）という構造で伝える。（例：20260219加納遥さん）

Q ピグマリオン（算数系）に通っているが、両方はきつい

理想の回答
- 「辞める必要はありません。こちらで土台を身につければ、ピグマリオンさんでの学びも相乗効果で伸びます」
- 「ただ、経済的に両方がきついのであれば、こちらの方が英語・運動・マインド含めてカバー範囲が広い。一旦こちらに集中して、落ち着いたらまた戻す選択もあります」

補足 「どちらか」を迫るのではなく、「こちらの方がカバー範囲が広い」と伝える。本人自ら「そっち休んでもこっちの方がいい」と判断してくれるのが理想。（例：20260220佐々木彩奈さん）

Q ベビーパーク（七田系等）と並行できるか。やりきれるか不安

理想の回答
- 「並行していただいて大丈夫です。内容が違います。ベビーパークは月例に沿った課題、こちらは思考力・探求力のスイッチとママの関わり方の設計がメインです」
- 「200本ありますが、お子さんに必要なのは絞られます。担任の先生と相談しながら選べるので、やりきれない心配は減ります」
- 「2人目のお子さんも入るなら、2人分の成長と考えられてコスパも良いです」

補足 講師自身のベビーパーク切り替え事例（お手伝いで握力→濃くかける→鉛筆）を出すと、並行可・内容が違うが腹落ちしやすい。（例：20260302森谷真由美さん）

11. アクセシビリティ・特別な事情

Q 視覚障害がある。会員サイトは使えるか

理想の回答
- 「基本的にテキストベースなので、PCの読み上げ機能で操作できると思います」
- 「画像やグラフは難しいかもしれませんが、講座の流れや教材の説明は全て文章になっています」
- 「入ってみないと分からない部分もあるので、クーリングオフ期間（8日間）の中で操作を確認してください。万が一操作が難しければご相談できます」
- 「Zoom講座では、スライドは全て読み上げますので、内容が伝わらないということはありません」

補足 視覚障害に限らず、操作面に不安がある方には「クーリングオフ期間で確認→ダメならご相談」の安全策を先に示す。（例：20260220佐々木彩奈さん）

12. 教材・コンテンツの更新

Q 教材は時々更新されるのか／1年後も同じか

理想の回答
- 「教材自体は基本的に同じです」
- 「ただ、お子さんの年齢を重ねることにできることが変わってくるので、同じ動画でも全く違った遊びになります」
- 「1年後にまた見直したいなと思ったら、月2,500円の継続プランで引き続き閲覧できます」

補足 「更新されない＝古い」と受け取られないように、「同じ教材でも年齢によって使い方が変わる」と価値を保つ。（例：20260220佐々木彩奈さん）

13. IQ・成果の測定

Q IQはいつ測るのか／講座内で測れるか

理想の回答
- 「講座の中ではIQの検査は行っていません。公的な機関に検査に行っていただく形になります」
- 「幼児教室に通われている方は、その教室でIQ検査がある場合もあるので、それを目安にされている方もいます」
- 「IQ200を目指すというより、8つの基礎力をバランスよく伸ばした結果、気になって測ってみたら200超えてた、という感じです」

補足 IQの数値にこだわりすぎない方向に誘導する。「IQ200はゴールではなく結果」というフレーミング。（例：20260220佐々木彩奈さん）

14. 他プログラム受講済み・予算制約

Q ママの気持ちを整える系の無料講座を受けた。教材がなくママの働きかけのみだったので、教材付きの講座を探している

理想の回答
- 「こちらの強みは、ママのOSを変えるワークに加えて、200本以上の動画教材と担任による個別設計があることです。何をやればいいか迷わず、お子さんに合ったものを選べます」
- 「ママの気持ちを整える土台は共通。その上に、具体的な声かけ・アクティビティの設計が乗る形です」

補足 「教材等がなく、ママのみの働きかけ」の他講座と明確に差別化する。本人の「そこが違う」に応える形で価値を伝える。（例：20260302森谷真由美さん）

Q 発達科学コミュニケーション等に高額を払ったばかりで追加投資は難しい

理想の回答
- 「発達科学コミュニケーションは親の声かけを変えるもの。うちは親のOSアップデートに加えて子供向けの動画教材200本がある。レイヤーが違うので補完関係にあります」
- 「発コミュではサポートや相談ができないとおっしゃっていましたよね。うちは担任制でいつでも相談できます。その部分だけでも価値があると思います」
- 「分割なら月3.6万円で、お2人のお子さんで使えます。2人分で考えると月1.8万円です」

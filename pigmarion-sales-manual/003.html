## 旦那の5つのパターン

### パターン①：無関心型（最多・最難関）

| 項目 | 内容 |
|------|------|
| 口癖 | 「好きにしていいよ」「任せるよ」「ふーん」 |
| 本音 | 教育に興味がない。考えたくない。面倒。自分の仕事で手一杯 |
| 特徴 | 普段は口を出さないが、高額の出費が発生した瞬間に「え、いくら？」と初めて反応する |
| 厄介な点 | 反対はしないので「大丈夫かな」とママが油断する → 金額が出た瞬間にブレーキ |
| 代表的な家庭像 | 世帯収入800〜1,200万。パパは激務で帰宅22時以降。教育はママに丸投げ |

#### なぜ難しいのか

パンフレットや権威性を見せても「ふーん」で終わる。そもそも情報を読まない。ママが一生懸命プレゼンしても「いいんじゃない？でも高いね」で終わる。問題は「反対」ではなく「不在」。関心がないから論理も感情も刺さらない。

## パターン①攻略：無関心型

### なぜ権威性だけでは動かないのか

無関心型の旦那は、パンフレットを「読まない」。権威性（三笠書房出版、TEDx、16年の実績）を見せても、そもそも興味がないから目に入らない。これは「反対」ではなく「不在」なので、説得の対象にすらなっていない。

### 攻略の本質：「ママの変化」を見せつける

無関心型には**論理ではなく、体験**で動かす。具体的には：

> **ママが変わる → 子供が変わる → 家庭の空気が変わる → 旦那が「何やってるの？」と聞いてくる**

この順番でしか動かない。

#### セールスで言うこと

```
「旦那さまが"好きにしていいよ"タイプの場合、
 説得しようとするほど逆効果です。

 一番効くのは、ママが変わることです。

 実際の受講生の例ですが、
 ご主人は最初、完全に無関心でした。
 でもママがメソッドを始めて3ヶ月、
 子供が自分からお弁当を作るようになり、
 帰宅したパパに"見て！自分で作ったの！"と見せた。

 パパは"え、どうしたの？"と。
 そこからパパもボードゲームに参加するようになり、
 今では週末に家族でトランプ大会をやっています。

 ポイントは、旦那さまを説得するのではなく、
 "家庭の空気が変わったこと"を
 体験させることなんです。」
```

#### セールスでの具体的な提案

```
「旦那さまには、まず金額だけお伝えするのではなく、
 "私が変わりたいと思ったから、まず3ヶ月やってみたい"
 と伝えてみてください。

 "子供の教育に○○万円かかる"と言うと
 "え？"と反応しますが、
 "私自身が成長したいから始めたい"と言うと、
 反対しづらいんです。

 これは"子供のため"ではなく
 "ママ自身のため"というフレーミングです。

 実際、受講生の旦那さまの多くは、
 "ママが楽しそうにしている"のを見て
 初めて関心を持っています。」
```

#### マーケティングで仕込むこと

| 施策 | 内容 | 効果 |
|------|------|------|
| オプチャ投稿 | 「旦那が無関心だったけど、子供が変わったら"何やってるの？"と聞いてきた」エピソード | 同じ状況のママが「うちもそうなるかも」と希望を持つ |
| ライブ内 | 「パパの説得は不要です。ママが変われば家庭が変わり、パパは勝手に巻き込まれます」と明言 | 「旦那を説得しなきゃ」というプレッシャーを外す |
| LP | 「パパが戦力に変わった」事例を追加。特に「最初は無関心だったパパ」のビフォーアフター | 無関心型の旦那を持つママに直接刺さる |
| 説明会LP | 「おひとりでのご参加OK」を明記 | 「旦那に相談してから」のブレーキを先回りで外す |

#### ツール：「ママの決意レター」テンプレート

説明会後にママが旦那に送る用のテンプレートを用意する。

```
【旦那さまへのメッセージ例（無関心型向け）】

「今日、子育ての個別相談に参加してきました。

 子供の能力を診断してもらって、
 〇〇は探究心がすごく強いけど、
 コミュニケーション力にもう少し伸びしろがあるって分かりました。

 1日10分の関わり方を変えるだけで
 この部分が伸びるんだって。

 私自身の声かけを変えたくて、
 3ヶ月のプログラムに参加したいと思ってます。

 金額は○○万円。
 分割だと月○○円。

 効果がなかったらやめるし、
 まず3ヶ月だけ試させてほしい。

 〇〇（子供の名前）が自分から動く子になったら、
 パパも楽になると思うよ。」
```

**ポイント：**
- 子供の具体的な診断結果を入れる（数値で見せることで「なんとなく良さそう」ではなく「根拠がある」に変わる）
- 金額を正直に書く（隠すと後で問題になる）
- 「3ヶ月だけ試させてほしい」でリスクを下げる
- 最後に「パパも楽になる」で旦那のメリットに触れる

## パターン②攻略：保守型（「普通でいい」）

### なぜ「普通」が最大の敵なのか

保守型の旦那は、自分が「普通に」育ったことを根拠にしている。「俺は公立で普通に育ったけど、ちゃんと就職できたし家族も養えてる」。この成功体験が最強の壁。

### 攻略の本質：「普通の基準が変わった」ことを伝える

旦那の「普通」は20〜30年前の基準。今の「普通」は全く違う。ここを論理で崩す。

#### セールスで言うこと

```
「旦那さまが"普通でいい"とおっしゃる場合、
 こうお伝えしてみてください。

 "30年前の普通と、今の普通は違うよ。
  パパの時代は大学を出て大手に入れば安泰だった。
  でも今は大手でもリストラされるし、
  AIに仕事を奪われる時代。

  私たちの子供が大人になる20年後、
  今の"普通"では通用しない可能性がある。

  別に天才にしたいわけじゃない。
  ただ、自分で考えて自分で動ける子にしたい。
  それが今の時代の"普通"だと思う。"

 保守型の旦那さまには、
 "頑張りすぎ"ではなく"時代に合わせてる"
 というフレーミングが効きます。」
```

#### セールスで使える具体データ

```
「旦那さまに見せる用のデータとして、

 ・闇バイトの47%が25歳以下
```

金銭管理型は内容には興味がある（または反対しない）が、金額で止まる。「高い」ではなく「今は出せない」が本音。

### 攻略の本質：比較と分割で「出せる」に変える

#### セールスで言うこと

```
「旦那さまが金額を気にされる場合、
 比較で伝えるのが効果的です。

 "インターナショナルスクール：年間200〜300万円
  習い事5つ掛け持ち：月5〜8万円（年間60〜96万円）
  高額英語教材：80万円

  しかもインターや習い事は
  "通っている間だけ"の効果ですが、
  このメソッドは"ママのOS"が変わるので、
  一度身につけたら一生使えます。

  投資ではなく、保険だと思ってください。
  子供の脳のゴールデンエイジは8歳まで。
  この期間を逃すと、あとからいくらお金をかけても
  同じ効果は得られません。"」
```

## 複合パターン攻略：「無関心型 × お金がない」

旦那対策の最難関。無関心型の旦那に対して「ママが自分のお金で投資する」が使えないケース。専業主婦で自由になるお金がない、世帯収入が中間層で教育費の大型支出には旦那の承認が必須。

### なぜ既存の攻略法が効かないのか

| 既存の攻略法 | この複合パターンで効かない理由 |
|------------|---------------------------|
| 「ママが自分で決めて投資する」 | お金がない。旦那の承認なしに払えない |
| 「ママが変われば旦那は巻き込まれる」 | 正しいが、変わるための投資ができない。鶏と卵 |
| 「ファクトシートを見せる」 | 無関心だから読まない |
| 「えみこ先生と15分面談」 | 無関心だから「面談？いいよ別に」 |
| 「分割で月○○円」 | 月額でも「何に使ってるの？」と聞かれる |

### この複合パターンの本質

ママは「やりたい」、旦那は「興味ない」、お金は「旦那の承認が要る」。つまり**ママ一人では動けないし、旦那を動かす手段もない**という詰み状態。

この詰みを崩すために必要なのは**3つのレバー**。

---

### レバー1：旦那の世界に存在する「トリガー」を使う

無関心型の旦那が唯一反応するのは、**自分の世界から来る刺激**。ママの世界（教育、子育て、ライブ）は完全に別世界なので響かない。旦那の世界に存在するトリガーを特定し、それを経由させる。

#### トリガー①：子供が「パパ見て！」をする瞬間

旦那は教育に無関心だが、子供には関心がある（あるいは、子供に喜ばれたい欲求がある）。

**具体的なアクション：**
メソッドの「スイッチ遊び」を1つだけ、ママが無料で実践する。たとえばお風呂で「浮くかな？沈むかな？」をやる。子供がハマったら、週末にパパとお風呂に入るときに子供が自分から「パパ、これ浮くと思う？」と聞く。

この瞬間が、旦那にとっての「何それ？」のスイッチ。ママからの説明は響かないが、子供からの「パパ見て！」には反応する。

**セールスで言うこと：**

```
「旦那さまが無関心な場合、
 まずママだけで1つだけ試してください。

 お風呂で"これ浮くかな？沈むかな？"と
 お子さんに聞くだけです。お金はかかりません。

 お子さんがハマったら、
 週末にパパとお風呂に入ったときに
 子供が勝手に"パパ、これ浮くと思う？"と聞きます。

 そのときの旦那さまの顔を見てください。
 "何それ？面白いね"と言った瞬間、
 それがスイッチです。

 そのタイミングで初めて
 "実はこういうメソッドがあって…"と切り出す。
 子供の体験を通じて旦那さまに入るんです。」
```

#### トリガー②：旦那の「仕事脳」にヒットする言葉

無関心型の旦那でも、仕事の文脈なら情報を処理する。「子育て」の話は聞かないが、「投資」「ROI」「リスクヘッジ」の話は聞く。

**具体的なアクション：**

```
【ママが旦那に言う台本（ビジネス脳向け）】

「ねえ、ちょっと聞いてほしいんだけど。

 今うちが子供の習い事に月○万円かけてるじゃん。
 年間だと○○万円。これ、8歳までだとして
 あと○年で総額○○万円になるんだよね。

 で、その投資のリターンって
 正直よく分かんなくない？

 今日聞いた話なんだけど、
 子供の脳の80%が8歳までに完成するらしくて、
 その時期に"何を"やるかで
 残りの人生のベースが決まると。

 ノーベル賞の経済学者が
 "幼児期の教育投資のROIが人生で最も高い"
 って言ってて、
 それを体系化したメソッドがあるんだけど、

 月○○円で、今の習い事より安い。
 3ヶ月で効果が出なかったらやめていい。

 投資判断として、どう思う？」
```

**なぜ効くか：** 「子育て」を「投資判断」に変換している。無関心型の旦那は子育ての話は聞かないが、「お金の使い方」の話は聞く。しかも「今かけてる習い事のROIが不明」という問題提起から入ることで、旦那の「損したくない」本能を刺激する。

#### トリガー③：旦那の友人・同僚からの情報

ママからの情報は「また何か始めるの？」だが、旦那の同僚が「うちの嫁がやってるんだけど、子供がすごく変わった」と言ったら態度が変わる。

**これはマーケティングで直接操作できない**が、間接的に仕込むことはできる。

- オプチャ内で「旦那さまの職場の同僚にも広がっています」的なエピソードがあれば共有
- 「とことこ」さん（唯一の父親参加者）の存在を活用。「パパも参加してくださっている方がいます」

#### トリガー④：学校・保育園からの指摘

最も強力な外圧。「先生に○○と言われた」「療育を勧められた」「成績が落ちた」。外部の権威からの指摘は、無関心型の旦那を一瞬で動かす。

**セールスで言うこと：**

```
「もし旦那さまに切り出すタイミングを
 考えているなら、
 次の面談や通知表のタイミングが1つの手です。

 "先生にこう言われた"
 "通知表がこうだった"
 "周りの子と比べてこの部分が心配"

 旦那さまが"まずいかも"と感じた瞬間に、
 "実はこういう方法があるんだけど"と出す。

 無関心だった旦那さまが
 急に真剣になるのは、
 "外からの指摘"があった時だけです。
 そのタイミングを逃さないでください。」
```

#### トリガー⑤：旦那自身の「子供時代の後悔」

「俺は普通に育ったからいい」と言っている旦那でも、心の奥で「もっとこうだったら」という後悔を持っている場合がある。「英語がもっとできたら」「もっと自分に自信があれば」「もっと主体的に生きていれば」。

**セールスで言うこと：**

```
「旦那さまに"あなたはどう？"と
 聞いてみるのもひとつです。

 "もし子供の時にもっと○○できてたら、
  今の仕事もっと楽だったと思う？"

 英語、プレゼン力、自分で考える力…
 旦那さま自身が"これがあったらな"と
 感じたことがあれば、
 それが子供への投資の理由になります。

 "俺みたいになってほしくない"は
 最強のモチベーションです。」
```

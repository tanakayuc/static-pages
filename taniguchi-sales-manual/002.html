EV7固有の訴求軸（面談で必ず接続する）

  1. 家・家族運気 … 間取り・方位・気の流れが家族の健康・関係・金運に直結する、と本人の悩み語りに結びつける。
  2. 子の進路・教育費 … 受験・不登校・教育費101万級の出費がある家庭では、親の基地包囲と「受験シーズンだけの過ごし方」を先に設計する。
  3. マスターコース50万円前後 … 銀振・カード・分割の説明は、受験費・建て替え費のあとでも始められる時期設計とセットで話す。
  4. 基地包囲 … 開運は行動。家だけでなく、親が取る行動（掃除・配置・朝晩の習慣）を一枚で示す。
  5. 足湯21日・30日チャレンジ … 高額の前に「まず21日」を具体行動として承認し、効果の入口を体感ベースで語る。
  6. 押し売りしない … 即決を迫らない姿勢は多くの分析で適切評価。ただし「決めるのはあなた」と行動の覚悟確認は別軸で入れる。
  7. 時期ずらし … 受験終了後・ボーナス後・建て替え一段落後など、開始月を提案しないと否決に直落しやすい（吉川・中原・藤原系）。

使い方

  • 個別前: 申込フォームの悩みからペルソナを当て、該当する成功例だけ読む。
  • 個別後: 分析の「マニュアル還流メモ」を本ファイルに1行追記する（原本・分析が揃っている場合）。
  • 価格の前に「いつから・何を・どのくらい動くか」を言語化してからマスター50万へ入る。

  • 出典: 20260523_岡村愛弓_永吉_分析.md（保留）

成約パターン（40件分析からの型）

  1. 申込動機（家・家族・健康）をそのまま受け、基地包囲・足湯21日で「今週からできる行動」に落とす。
  2. マスター／ゴールド／プラチナの違いを、相手の時間と家の状況で1コースに絞る。
  3. 価格の前に支払い方法（カード・分割・開始月）を選択肢として並べる。
  4. 「決めるのはあなた」「無理な勧誘はしない」と言いつつ、覚悟（週あたりの行動時間）を軽く確認する。
  5. 成約後の初動（決済・Chatwork・初回勉強会）を同じ面談で言語化する。

否決パターン

  1. 教育費・受験・不登校で家計が枯渇し、50万が心理的に「もう一つの受験費」に見える。
  2. 「本当に変わるか」への回答が講座説明の繰り返しになり、事例・数値・期間の約束がない。
  3. 長い雑談・共感のあと、講座説明と価格が後半に圧縮され、納得の時間が足りない。
  4. 配偶者・家計の合意が取れていないまま当日決済を求める（または逆にフォローなしで終了）。

保留パターン

  1. 前向きだが分割・開始月・配偶者確認が残る（岡村・味村型）。
  2. 効果は信じたいが、過去の風水・講座経験で慎重（事例と保証的フォローが必要）。
  3. 次回面談日と持ち帰り資料（支払い表・夫向け1枚）を必ずセットで約束する。

17ステップとの接続（EV7向け短縮メモ）

  • ステップ5〜6: 家・子・金のギャップを顧客の言葉で整理（表の申込項目以上を引き出す）。
  • ステップ8: 真因は「家の気」だけでなく、親の行動とタイミング（時期ずらし）の二層で示す。
  • ステップ10〜11: 反論は熱意ではなく、同ペルソナ事例と21日行動で処理する。
  • ステップ15〜16: 押し売りしない＝即決強要しない。保留と否決の境界は「次の一手が日付付きか」。

議論の流れ・全体構成

前半は田中からの近況共有（AI制作フロー・本出版・開発プロジェクト等）、後半はsawaのメタ広告アカウント問題の対策協議が中心。岡田さんがビジネスマネージャーの構造を解説しながら、今後の広告運用方針を整理した。

1. 田中からの近況共有

LP制作フロー（AI完全内製化）

案件LP（藤田かこさんの脳科学カウンセラー養成講座）は自分で作り直した。受講生が作ってきたLPが「やかましい」と判断し、その場でChatGPTで全部作り直してフィードバック済み。

現在確立しているAI制作フロー：
- Cursor → ライター（コピーライティング）
- ChatGPT → デザイナー（画像・バナー生成）
- Claude Code → コーダー（HTML/CSS実装）

以前はディレクター・ライター・デザイナーを外注していたが、全員不要になった。唯一残っているのはCanvaやUTAGEの細かい操作担当（従来報酬の3分の1程度で週数回）のみ。

LP制作でのChatGPTとのやり取りのコツとして「感覚を数字で言語化させる」技を紹介。「縦が長い」→「934ピクセルにしてほしい」など、自分が言葉にできないもやもやをAIに補足させながら進めると精度が上がる。

メタAIを広告審査の事前相談に活用するのが有効という見解も共有。

チャレンジローンチ完全ガイドブック出版準備

ムック本（B4サイズ）の執筆完了。プロモーション資料をそのまま書籍化した形。図解制作（ChatGPT）が残っているが文章は完成。

並行してDTPツールをClaude Codeで開発中。ラスクル等の印刷会社APIと倉庫（配送）APIを連携させて、ほぼゼロ円で紙の本を自動配送できる仕組みを目指している。目標コストは100部で10〜20万円程度（印刷・配送費込み）。

「広告が通らない業界で本のLPを使って広告を通す」という戦略の中核になる施策。

その他開発プロジェクト（趣味枠）

子供の写真・動画共有アプリ「みてね」のデータを取得し、YouTuberの旅行先をGoogleマップ上にマッピングするツールを開発中。家族の思い出の旅行記録ビジュアル化が目的。

家計簿ツールも開発済み。1月分の給与支払いが漏れていたことをツールが検知した。

2. sawaのメタ広告アカウント問題

発生した問題の経緯

紫（むらさき）さんのビジネスマネージャーが広告アカウントの上限（10個）に達しており、新規作成不可の状態。

別の手として、瀬太郎さんのアカウントに空きが1つあったため新規作成を試みた。しかし：
- 動画リール広告がアップロードできずに止まる
- 静止画に切り替えたら一旦審査通過・アクティブ化
- すぐに停止

その後、sawaの個人アカウントが一時停止に。「セルフィー動画（右・上・左と顔を動かす本人証明）→パスポート提出」という手順を経て個人アカウントは数時間で復活。ただし広告アカウントの制限は継続。

sawaの個人アカウントはFacebookページが作れない状態になっている。

今後の対処方針

高橋先生（高見さん）のアカウントに招待してもらい、その既存のFacebookページ・インスタを使って広告運用を開始する方向で進行中。担当者に招待を依頼済みで返答待ち。

並行して、紫さんの既存の（使っていない）広告アカウントを次世代型セールスの運用に再利用する。別案件のピクセルデータが残っていても、今のメタはクリエイティブ内容・テキスト訴求を読んで配信先を決めるため、過去ピクセルデータの影響は小さい。

3. ビジネスマネージャー構造の整理（岡田さん解説）

メタのアカウント構造についての確認事項を整理：

  • ビジネスマネージャーが最上位の「親」
  • その下にFacebookページ・インスタアカウント・広告アカウント・ピクセルが並列で紐付く（階層ではなく放射状）
  • 個人アカウントにつきビジネスマネージャーは2つまで
  • 新規ビジネスマネージャーは最初広告アカウントが1つしか作れない（運用実績が積まれると上限が増える）
  • 予算枠（Spending Limit）はビジネスマネージャー単位で管理

インスタとFacebookページの紐付けについて：
- Facebookページにインスタアカウントを1対1で紐付ける
- どちらか一方でも広告は出せるが、両方あるのが理想
- 最近は紐付けなしだと広告が出せないケースが増えてきている

新規案件の広告アカウント問題

新規ビジネスマネージャーで立ち上げる場合、予算上限が低い状態からスタートするため「バーン！と踏めない」問題がある。この状態でローンチのような高速消化が必要な案件に対応するのは難しい。

対策として：
- 最初から月30万円規模のやさしい案件から始めて信頼を積む
- 拡散広告（インプ稼ぎ）よりもシンプルなCV広告を細々と流して地ならしする方が有効
- クライアントには「最初は1社だけでなく他の媒体も並行利用してほしい」と調整が必要

4. 今後の広告運用方針

クライアントのビジネスマネージャーに広告運用者として入れてもらう形を基本方針とする。自分たちでアカウントを作って代行する形は、審査・インスタ紐付け・予算枠の問題で限界が来ている。

クライアント側に必要な準備として：
- ビジネスマネージャーの作成
- Facebookページの作成・インスタアカウントの紐付け（インスタパスワードが必要なため本人にやってもらうしかない）
- 広告運用者としてzaridのメンバーを招待

今後の新規案件立ち上げ時はこのセットアップを最初のミーティングで一緒にやってしまうのが現実的。

5. 次世代型セールスと45分セールステンプレートの関係

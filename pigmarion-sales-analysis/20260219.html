個別説明会 セールス担当フィードバック：20260219 加納遥さん

本ドキュメントは、セールス原本「20260219個別説明会with加納遥さん」に基づく、個別説明会担当者向けのフィードバックです。

参照：セールス原本 20260219個別説明会with加納遥さん.md／ターゲット情報.md／個別説明会_セールス成功失敗マニュアル.md／質疑応答の理想回答集.md／オプチャ参加者分析_ペルソナ_ニーズ.md

前提：この回の相手像

  • 東京在住、専業主婦、4歳女の子（幼稚園年少）
  • 1歳からトイズアカデミー（幼児教室）に通い、IQテスト150。教室から「これから下がる」と言われている
  • Can do listとワークに興味を持ったが、幼児教室とかぶる部分もある
  • トイズアカデミーを辞めるつもりはない
  • ここ2年は不妊治療に費用をかけており、今は出せない
  • 結果：否決

ペルソナ分類

既存ターゲット情報のどのペルソナにも完全には当てはまらない、新しいタイプ。

「現状うまくいっている（ように見える）ママ」

  • IQ150という高い数値が出ている → 「うちの子は大丈夫」の安心がある
  • 幼児教室に継続して通えている → 投資はしている、方向性に満足
  • 不安の質が「今困っている」ではなく「将来下がるかもしれない」の予防型

これは、既存の成功例（大橋さん・水野さん）と根本的に状況が異なる。大橋さんは「何もやってこなかった焦り」、水野さんは「前回諦めた未練」。加納さんは「今はうまくいっている。でも将来が不安」。危機感のレベルが低い。

この回の否決要因の構造

要因 詳細 対処可能性
金銭的制約（不妊治療） ここ2年の不妊治療で資金が限られている。これは本当の制約 分割提案・開始時期の調整で一部対処可能だが、本質的には難しい
現状への満足度が高い IQ150が出ており、教室にも満足。「もっと」の欲求はあるが「今のままではまずい」の切迫感がない 対処すべき本丸
幼児教室との重複感 Can do listやワークがかぶると感じている → 「追加で払う価値」が見えにくい 差別化の伝え方で対処可能
「辞めるつもりはない」の防御壁 教室を否定されたくない心理 → こちらが教室を否定すると逆効果 教室を肯定したうえで「足りないもの」を示す

「現状満足型」への危機感アプローチ（振り返り＋提案）

担当者が「現状に満足している方にもっと危機感を持ってもらえるアプローチがなかったか振り返ります」と書いている。この層に対する危機感の作り方を整理する。

やってはいけないこと

NG 理由
「IQ150は大したことない」と否定する 自分の選択を否定された→心が閉じる
「幼児教室では足りない」と直接言う 「辞めるつもりはない」と宣言している人に逆効果
「下がりますよ」と恐怖だけで煽る 教室から既に言われている。同じことを言っても「知ってます」で終わる

有効なアプローチ

1.「IQが高い子"だからこそ"のリスク」を見せる

IQ150の子供は認知能力が高い。しかしその子が小学校に入ったとき、「授業がつまらない」「周りと話が合わない」「先生の言うことが退屈」となり、不登校・適応困難・意欲喪失になるケースがある。これは大橋さんの上のお兄ちゃんと同じ構造。

トーク例：
「IQ150、素晴らしいですね。でもIQが高い子って、実は小学校で一番苦しむリスクがあるんです。授業がつまらない、周りと話が合わない、結果としてやる気がなくなる。うちのスクールにも、IQが高いのに不登校になった子がいます。IQが高いからこそ、EQ（社会性・共感力・自己制御）を同時に育てないと、バランスが崩れるんです。トイズアカデミーさんでIQを伸ばしているなら、うちがEQの部分を補完する形が最強だと思います」

→ 教室を否定せず、「足りないピース」を示す。

2.「IQは下がる」を「なぜ下がるか」の構造で伝える

教室から「下がる」と言われているが、「なぜ下がるか」は説明されていない可能性が高い。

トーク例：
「教室の先生が『下がる』とおっしゃったんですね。実はIQが下がるのは、年齢が上がると周りも伸びてきて相対的に差が縮まるからなんです。でも本当に大事なのは、IQの数値ではなくて、その子が"自分で考えて動ける力"を持っているかどうか。IQが150あっても、指示待ちの子になったら意味がない。逆にIQが120でも、探究心と自己効力感がある子は自走できる。うちのメソッドは、IQの数値ではなく"自走する脳"を作るんです。トイズアカデミーさんとは、そもそもレイヤーが違います」

→ 「下がる」を「構造」で説明し、「だから別のアプローチが必要」に接続する。

3.「4歳→小学校入学」のタイムラインを個別化する

4歳の女の子ということは、小学校入学まであと約2年。

トーク例：
「4歳ということは、小学校入学まであと約730日。その730日で、IQの土台の上にEQの柱を立てられるかどうかで、小学校6年間が全く違うものになります。幼児教室でIQを伸ばしている今だからこそ、同時にEQを育てるのが一番効率的なんです。別々にやるより、同じ時期にやる方が相乗効果があります」

→ 「今やっていることを活かすために、今足す」というフレーム。

4. 不妊治療の金銭制約への対応

不妊治療は否定できない。しかし「今は出せない」が「永遠に出せない」ではない。

トーク例：
「不妊治療、大変ですよね。お金のことは本当に大事なので、無理にとは言いません。ただ一つだけお伝えしたいのは、不妊治療の結果がどうなっても、今いる4歳の娘さんのゴールデンエイジは待ってくれないということです。分割でのお支払いや、開始時期をずらすことも可能ですので、もし"やりたいけどお金のタイミングだけ"ということであれば、一度ご相談いただけませんか？」

→ 「不妊治療 vs 教育投資」の二択にせず、「両立できる方法」を提案する。

成功失敗マニュアルへの追記提案

新しいパターン：「現状満足型」の否決

項目 内容
パターン 既に幼児教室等に投資しており、一定の成果（IQ高い等）が出ている。メソッドに興味はあるが「追加で払う価値」が見えにくく、既存の投資を辞めるつもりもない
対処 ①教室を否定しない。「すでにやっていることを活かすために足す」フレーム ②IQだけでは足りないリスク（高IQの子の小学校適応問題等）を構造で伝える ③「レイヤーが違う」→ 競合ではなく補完の位置づけ
注意 恐怖訴求が効きにくい層。「今困っていない」人に「将来困る」だけでは動かない。「今やっていることの効果を最大化するために」という"最適化"の文脈の方が刺さる

質疑応答の理想回答集への追記提案

新カテゴリ：「既存の教育投資との重複」

Q 幼児教室とかぶる部分がある／今の教室を辞めるつもりはない

理想の回答：
- 「辞める必要はまったくありません。むしろ〇〇教室でIQを伸ばしているなら、その土台の上にEQ・非認知能力を乗せるのが最も効率的です」
- 「トイズアカデミーさんは認知能力を伸ばすのが得意。うちは非認知能力とママのOSアップデートが得意。レイヤーが違うんです」
- 「Can do listやワークにかぶりを感じたかもしれませんが、うちのメソッドの本質はワークの中身よりも"ママの関わり方の設計"の部分。教室では教えてくれない部分です」

（例：20260219加納遥さん）

まとめ

この回の最大の学びは、「現状に一定の満足がある層」への危機感の作り方。今回のプロモーション参加者の中には、オプチャ分析でも「幼児教室に通っているが行き詰まっている層」が最多だったが、加納さんのように「行き詰まっていない層」も一定数いる。この層には恐怖訴求ではなく、「今やっていることの効果を最大化する」「IQが高いからこそEQが必要」という最適化フレームが有効。

金銭制約（不妊治療）は本質的な障壁であり、無理にクロージングする場面ではない。しかし「やりたいけどタイミングだけ」であれば、分割・開始時期の調整を提案し、接点を切らないことが重要。

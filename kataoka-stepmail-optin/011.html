# 📢 【AIコンサル】「型」を持って6ヶ月後に立っている自分を想像してみてください

Claude特化型AIコンサルタント
ナビゲーターの田中祐一です。

今朝の片岡さんのメール、
読んでくださったでしょうか。

7ステップ。

これを全部「自分の言葉」で
語れるようになると、
何が変わるのか。

今夜は、その「変化の絵」を
ご一緒に描きたいと思います。

「片岡さんが9年で1.8億円を作った方法」
動画で直接聞けます。

🔗 [動画ページURL]

＝＝＝＝＝＝＝＝＝＝＝＝＝＝
【6ヶ月で起きる、典型的な変化】
＝＝＝＝＝＝＝＝＝＝＝＝＝＝

僕がこれまで70業種以上の
プロモを支援してきて、

「型を渡された人」が
6ヶ月で辿る道筋は、
意外なほど似ています。

【1ヶ月目】
　名刺と自己紹介トークを準備。
　地元の交流会に1〜2回出る。
　まだ商談ゼロ。でも怖くなくなる。

【2ヶ月目】
　最初の商談が湧いてくる。
　「もう少し詳しく聞かせて」と
　次のアポが入り始める。

【3ヶ月目】
　最初の契約。
　50万円〜100万円のラインで決まる。
　ここで一気に景色が変わる。

【4ヶ月目】
　紹介で次の商談が入る。
　「うちの取引先にも紹介していい？」
　という声が来始める。

【5〜6ヶ月目】
　月の固定収入が積み上がる。
　月50万円ラインを超える。

　「会いに行ける範囲」だけで
　仕事が回り始める。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝
【「型がある人」と「ない人」の差】
＝＝＝＝＝＝＝＝＝＝＝＝＝＝

ここで、
ひとつ書き換えていただきたい
パラダイムがあります。

「成果は、努力量で決まる」

違います。

成果は、
型に乗った時間で決まる。

✔ 型がない努力10時間
　= ほぼ何も残らない

✔ 型に乗った努力10時間
　= 商談が1件、契約に近づく

同じ10時間でも、
半年後の場所が
まったく違います。

そして、
本講座でお渡しするものは
全部「型」です。

✔ 名刺の型
✔ 自己紹介トークの型
✔ ヒアリングの型
✔ AI社員デモの型
✔ 提案書の型
✔ 契約書の型
✔ 反論への切り返しの型

「自分でゼロから考えなくていい」

ここに、
最大の時短があります。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝
【本業との相乗効果】
＝＝＝＝＝＝＝＝＝＝＝＝＝＝

今このメールを読んでくださっている方の中には、
すでに本業をお持ちの方が
多いはずです。

✔ 社労士として顧問先がある
✔ キャッシュフローコーチとして契約がある
✔ 経営コンサルとして契約がある

そういう方にとって、
このAIコンサルタントは
「副業」ではありません。

「本業の単価を引き上げる入口」
です。

例えば社労士なら、
「AI社員で勤怠を仕組みに変えませんか」
という入口から、

就業規則・労務相談まで
連動して取れる。

ここの設計図は、
体験セミナーで
具体的にお見せします。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝
【動線はオープンチャットだけ】
＝＝＝＝＝＝＝＝＝＝＝＝＝＝

体験セミナーの日程公開も、
申込フォームの配布も、
特典の受け取りも、

質問への回答も、
全部LINEオープンチャットの中だけで
動かしています。

✔ 各都道府県3名までの地域枠
✔ 開催枠は数日中に埋まり始めます

メルマガだけで読んでいる方は、
そもそも申込フォームに到達できません。

オープンチャットに入っている方は、
枠を確認しながら、
その場で申し込めます。

数分の差ですが、
6ヶ月後の場所が変わります。

まだ動画を見ていない方は、
今夜のうちに10分だけ。
動画を見終わった先で、
オープンチャットへの案内が待っています。

1. 案件概要

  • 案件: 「100万マイラーの新常識」5日間無料トレーニング（講師: 増田和彦）のオプトインLP
  • 目的: メールアドレス登録（オプトイン）。広告・SNS流入のコールド層向け
  • ターゲット: 50〜60代の会社員・パート・主婦（6割女性）。旅行好き・マイルに関心・物販未経験
  • トーン: ロジックと数字で信頼を作る左脳型。高級感はあるが品がある（ネイビー×ゴールド）。キラキラ・虹色・装飾過多は禁止

2. 技術要件（必須）

  1. HTML 1ファイル完結（UTAGEのraw_html貼付を想定）。DOCTYPE〜閉じタグまで完全なHTML
  2. CSSは <style> インライン1ブロックのみ。外部CSS禁止
  3. JavaScript禁止（CTAボタンは <a href="#optin-form"> のアンカーリンクでOK。フォーム自体はUTAGE側が後で差し込むため、id="optin-form" の空セクションを最下部CTA直前に置く）
  4. フォント: Google Fonts Noto Sans JP（400/700/900の3ウェイトまで）。preconnect + display=swap
  5. モバイルファースト1カラム。コンテンツ最大幅 640px 中央固定。PCでは左右に背景色が残る
  6. 画像は全箇所プレースホルダでよい（後述§5）。width/height属性必須、FV以外は loading="lazy" decoding="async"、FVのみ loading="eager" fetchpriority="high"
  7. HTML 50KB以下目標

S6｜共通の敵

見出し: なぜ、今までのマイル術では届かなかったのか。

世の中で語られているマイルの貯め方は、ほとんどがこの3つです。

よくあるマイル術 続かない理由
クレカ入会キャンペーン 一度きりで終わる。カードの枚数にも限界
ポイントサイトの案件まわり 案件が枯れたら終わり。単発の積み上げ
紹介・アフィリエイト 発信力のある人にしかできない

共通点は、「再現性」がないこと。
来月も、来年も、同じペースで貯まり続ける保証が、どこにもありません。

毎月、同じ仕組みで、お金を稼ぎながらマイルを貯め続ける。

メディアも、発信力も、特別な才能もいりません。
普通の会社員が、今日から始められる方法です。

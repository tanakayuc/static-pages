1. 忙しさ・限界

Q もう忙しい。これ以上増やせない

理想の回答の方向性:

  • 「忙しさが成果の天井になっている」ことを一度言語化してもらい、次のステージの課題として正常化する。
  • 時短のテクニックだけでなく、「何を手放すか／何を仕組み化するか」の二択に落とす。

実例メモ: 吉岡理恵さん回（真のニーズが忙しさへの不安で前に進めない。ローンチ実績はあるが次のステージの悩みとして扱う。雑誌・出版オファーの見分け、顔出し最小化、キャッチコピーで検討違いを減らす論点は分析 20260306_吉岡理恵_五ヶ山昌代_分析.md Version 2.0 のシーン別を参照）。

2. 売上はあるが不安・プライドが高い

Q 表向きはうまくいっているが、実際は苦しい

理想の回答の方向性:

  • 承認は事実ベース（本・実績・発信の継続など）に限定し、同じブロックで市場・同業・次のレベルの課題を一枚入れる。
  • 褒めだけで終わらせず、痛みと機会損失を同じ尺で渡す。

実例メモ: 酒井佑佳さん回（出版や表向きの成功と、現金の苦しさのギャップ。不成約後の反省として「褒めと痛み」のバランスが言語化されている。理想の一文は「承認の直後に、同じブロックで機会損失を一枚入れる」。分析 20260310_酒井佑佳_五ヶ山昌代_分析.md Version 2.0）。

3. 知識・学習は多いが結果に結びついていない

Q 学んだけど売上・講座化に繋がらない

理想の回答の方向性:

  • 「点の知識」を「線の仕組み」に変える作業であると定義し、上部承認だけにしない。
  • 「何でも知っている」発言が出たら、否定せず、自分で課題に気づく問いに切り替える。

実例メモ: 市村睦美さん回（学んできているが結果に結びついていないアンケートを起点に深掘り。分析 20260310_市村睦美_三浦美幸_分析.md Version 2.0）。石井和美さん回（「なんでも知っている・やっている」言葉が多い。現場の稼働と来店から月次天井を計算し、公式LINE・導線の一点突破へ矢印を返す。分析 20260304_石井和美_五ヶ山昌代_分析.md Version 2.0）。出雲光さん回（学習歴が長く他塾でフォロー不足を経験、発言が多方向。無理に畳まず記録し難易度をチーム共有。分析 20260310_出雲光_五ヶ山昌代_分析.md Version 2.0）。

4. 他コンサル・起業塾・ネットワークとの兼ね合い

Q いま別の塾・ネットワークを優先したい

理想の回答の方向性:

  • 優先順位を否定せず、「いつ・何ができたら再検討か」を一文で合意する。
  • 複数塾に行っているなら、「全部やり切る」より「どれを主軸に成果を出すか」を問う。

実例メモ: 杉山敦子さん回（ネットワークの学び期限を優先し自ビジネスは後回し。冒頭で「ここではMLMの組織づくりの話は扱わない」などスコープを一言切ると摩擦が減ることがある。分析 20260313_杉山敦子_五ヶ山昌代_分析.md Version 2.0）。佐野恵子さん回（複数起業塾を使いこなしてからアカデミー、の意思を変えきれず。分析 20260317_佐野恵子_五ヶ山昌代_分析.md）。野口由美子さん回（他コンサルや金額感の比較が出る場面でも、安易に下げず本気の境界を示して成約。家族・内緒と重なるときは境界の一文を短く言い切る。書き起こし 20260324_野口由美子_五ヶ山昌代.md、分析 20260324_野口由美子_五ヶ山昌代_分析.md Version 2.0）。

Q また学ぶのか／学び疲れしている

理想の回答の方向性:

  • 「情報を増やす」のではなく「設計を一つ決める」個別だと枠を先に示す。
  • 複数塾と並走しているなら、主軸を一つに絞る質問へ寄せる。

実例メモ: 佐野恵子さん回（五ヶ山担当。分析 20260317_佐野恵子_五ヶ山昌代_分析.md Version 2.0）。

Q MLMに在籍していて、自ビジネスとどう両立するか分からない

理想の回答の方向性:

  • 在籍を否定せず、今日の個別のスコープ（自ビジネスの導線・講座化）に絞る。
  • 2年コミット等の拘束が先に立つなら、再提案の条件を日付つきで残す。

実例メモ: 森下美枝子さん回、高橋由美子さん回（和田・不成約寄りの論点の宝庫。分析 20260313_森下美枝子_和田めぐみ_分析.md、20260317_高橋由美子_和田めぐみ_分析.md Version 2.0）。

5. 家族・配偶者・内緒

Q 配偶者に言えない／相談はするが決められない

理想の回答の方向性:

  • 「相談は共有、決断は本人」に戻す。
  • 内緒で進める話が出たら、必ず社内ルール・コンプライアンスと照合し、本人のリスク説明をセットにする（マニュアルだけで完結させない）。

実例メモ: 池田記代さん回（夫に相談しても決めるのはのりよさん、の型。分析 20260316_池田記代_三浦美幸_分析.md Version 2.0）。野口由美子さん回（高額被害・夫婦関係・内緒で売上を立てたいなど背景が多層。成約済み。書き起こしで論点の流れを追える。フォロー・コンプライアンスは運用ルールとセットで設計する）。

Q 高齢の本人が「息子（家族）に相談してから返答したい」と言った

理想の回答の方向性:

  • 家族決裁が見えた時点で、本人の覚悟と並行して家族用の一文を本人の口から作らせる。
  • 「◯◯さんに今日なんて伝えますか？」「何がクリアになれば家族もOKと言いそうですか？」を本人に言わせ、本人が自分の言葉で家族用トークを持って帰る状態を作る。家族側の反射的NOが出にくくなる。
  • 即決を追わず、返答期限の明瞭さと家族用トークの完成度で「保留の質」を上げる方向に切り替える。

実例メモ: (2026-04-23 追記 / 出典: 20260422_磯和なつ_五ヶ山昌代) 82歳・事業を息子に譲渡済・当日20:00期限で息子相談待ちの事例。シナリオで真剣モードに引き戻せた瞬間が成功の半分、家族トークの一文まで作り切れたかが残り半分。分析 20260422_磯和なつ_五ヶ山昌代_分析.md Version 1.0。

6. 返答保留・即決しない方針

Q 日付まで考えさせてほしい／即決はしない

理想の回答の方向性:

  • 期限を明確にし、その日までに必要な情報（支払い方法・学習開始時期）をリスト化しておく。
  • 「何事も即決しない」と決めている相手には、即決を迫らず、期限内の中間確認日を1つ置く。

実例メモ: 石井和美さん回（3/25返答待ち。分析 Version 2.0）。柏木肇さん・朝月希光さん回（3月末返答。柏木は分析 20260325_柏木肇_五ヶ山昌代_分析.md、朝月は 20260325_朝月希光_五ヶ山昌代_分析.md Version 2.0）。算命学等で外部判断待ちのときは、占いとビジネス判断を分けつつ「次のミニコミット」（資料送付日・中間確認日）を一つだけカレンダーで固定する。

Q 今日は説明だけだと思っていた／個別のゴールが違う

理想の回答の方向性:

  • 責めずに、今日の個別で決めたいこと（情報整理・次一手・申込の可否）をその場で言語化し直す。
  • 見込み客の言葉で「今日の終了イメージ」を一文合意してから本題に入る。

実例メモ: 佐野恵子さん回（濱野担当。書き起こし ## 15:00 前後の勘違い訂正。分析 20260313_佐野恵子_濱野りん_分析.md Version 2.0）。

7. 年齢・キャリア後半（第二の人生）

Q 定年後も働きたいが遅くないか

理想の回答の方向性:

  • 経験・信用・ネットワークを「オンラインで再投資できる資産」として言い換える。

実例メモ: 杉山敦子さん回（定年後いきいき）。柏木肇さん回（85歳まで働く意気・BtoBとBtoC）。

8. オンライン・集客・仕組み化

Q 対面しか信じられない／集客が不安

理想の回答の方向性:

  • 既存の顧客層・実績・発信履歴を起点に、「見える化の場所を増やす」話に落とす。
  • ホットペッパーや紹介だけに依存している場合は、チャネル追加の意味を具体化する。

実例メモ: 市村睦美さん回（SNS嫌・月1更新などからの打ち手）。佐野恵子さん回（ホットペッパー集客の話題が断りトークに）。福山道子さん回（集客ニーズ＋「6月までに60万」など期限付き数字。本部インセンティブ期限・自由な時間とお金をベネフィットに接続し成約。澤田担当）。

9. 体験会・未来の数字が出るまで今日は決めたくない

Q 〇ヶ月後の体験会（ローンチ）の結果を見てから決めたい

理想の回答の方向性:

  • 成果待ちの気持ちを否定しない。
  • 「今日決めること」と「あとで決めること」を分離し、例えば学びの枠だけ先に確保する・オプションを残す、など商品ルールの範囲で選択肢を提示する。
  • 完全に先延ばしになる場合は、次の接触日と判断基準をメモに残す。

実例メモ: 吉岡拓也さん回（3ヶ月後の体験会計画の結果待ちで不成約。コーチング層向けには「今日決めないこと」と「次の接触日・判断基準」をセットでメモに残す。分析 20260313_吉岡拓也_五ヶ山昌代_分析.md Version 2.0）。

10. 会社・代表として稟議したい

Q 個人ではなく会社として受講を検討したい／社内で返答したい

理想の回答の方向性:

  • 代表個人の納得と、会社の意思決定プロセスを分けて聞く。
  • 返答期限と、次の打合せに必要な資料（カリキュラム・費用・期間）を一覧で渡す。

実例メモ: 柏木肇さん回（会社として受講・3月末返答。代表個人の納得と稟議プロセスを分け、カリキュラム・費用・期間の一覧を渡す質問リストをテンプレ化するとよい。分析 20260325_柏木肇_五ヶ山昌代_分析.md Version 2.0）。

Q 別ルート（別プロデュース等）にも動いていて、優先が決められない

理想の回答の方向性:

  • 冒頭30分で「他に動いている申込・期限」を棚卸しし、今日の個別での優先順位を一文で合意してから商品説明へ。
  • 長尺でも後半で初めて「両方は無理」が出ないよう、先に順序を外に出す。

実例メモ: 吉田雅子さん回（和田・別申込と現金の手元なし。分析 20260313_吉田雅子_和田めぐみ_分析.md Version 2.0）。

11. 占い・第三者の判断を挟みたい

Q 算命学（占い等）の先生に聞いてから方向性を決めたい

理想の回答の方向性:

  • 価値観を否定しない。
  • 「ビジネス上の判断」と「気持ちの整理」を分け、いつまでに連絡するかだけ合意する。

実例メモ: 朝月希光さん回（算命学の方に聞いてから・3月末返答。多話と期限返答が重なるパターンの参照用。分析 20260325_朝月希光_五ヶ山昌代_分析.md Version 2.0）。

12. 方向性が多すぎて一貫しない（難易度判断）

Q やりたいことが次々変わる／発言に一貫性が感じられない

理想の回答の方向性:

  • 無理にクロージングしない選択肢も持つ。記録に残し、チームで難易度を共有する。
  • 伴走が必要なタイプとして、いったん期待値を下げた提案（別プロダクト・別タイミング）も検討。

実例メモ: 出雲光さん回（不成約・担当所感でビジネス難易度高と記録。ノーハーコレクター×会社員×本業希望×発言の多方向性×他塾フォロー不足の重なり。前半で「今日の終了イメージ」と「詰まり1つ」を合意しないと講義が膨らむ型。分析 20260310_出雲光_五ヶ山昌代_分析.md Version 2.0）。

13. 分割プラン・一括・カード説明（顧客保護と成約の両立）

Q 分割は心理的に嫌だが一括も厳しい／カードの「あとから分割」は本当にできるのか

理想の回答の方向性:

  • 現実制約を聞いたうえで、コースの組み替え・分割回数・初回の負担をセットで示す（柴田慶子さん回のプラチナ着地が手本）。
  • 「あとから分割」「保険をかけない方が通る」等はカード契約・返済計画次第で破綻しうるため、強い言い切りのあとに事務局フォローで無理のない返済を再確認するフローをチームで固定する。

実例メモ: 柴田慶子さん回（濱野担当・分析 20260304_柴田慶子_濱野りん_分析.md Version 2.0 のチェックリスト）。石井和美さん回（五ヶ山担当・一括と分割の心理の往復。分析 20260304_石井和美_五ヶ山昌代_分析.md Version 2.0）。作本美香さん回（シルバー提示でもコンサル必須感が強く支払インフラが止まる型。最小単位の見せ方を次回設計。分析 20260326_作本美香_三浦美幸_分析.md Version 2.0）。

Q ChatGPT に聞くのとコンサルに聞くのは何が違うのか

理想の回答の方向性:

  • 情報取得と、責任を持った設計・伴走を分ける。
  • 「今日の一手の名前」を一緒に決めるのがコンサル側の価値、と一文で枠を示す。

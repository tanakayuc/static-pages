  • 結果に自信が薄い・知識は多いタイプに対し、「上部の褒めを封印」「できる貯金」を意識するなど、テンポと承認の使い分けが明文化されている（市村睦美さん回。分析 20260310_市村睦美_三浦美幸_分析.md Version 2.0）。

繰り返し論点・伸びしろ

  • 池田さん回で「もっとカットできるところがあった」と自覚あり。無駄な往復を減らす編集が次の課題（成功失敗マニュアルにも記載済み）。
  • 成約案件でも、支払・初回決済期日など表側データのブランクが残りやすい。CRM突合をルーティン化すると分析と連動しやすい（各分析の共通指摘）。
  • ネガティブ・支払い不安が強い相手に対してシルバー等で寄せた際、「コンサルなしではイメージが持てない」壁に当たるパターンが報告されている（作本美香さん回。分析 20260326_作本美香_三浦美幸_分析.md Version 2.0）。次回は「コンサル相当をどう最小単位で見せるか」を事前設計しておくとよい。
  • 再ズームでペース管理を意識しつつも、1時間超えで相手の集中が切れた後に説明を畳まず進め、不成約に至った反省が出ている（小川健司さん回）。疲労サインでオファーを切り、必ず再日程に寄せる。
  • 決済インフラ（カード枠・決済手段）がボトルネックになる案件では、トーク以前の制約として切り分けて記録するとよい（大原千里さん回）。
  • 長尺・ネットワーク併走の見込み客では、分割説明の前に「今月動かせる現金」を一度数字で確認し、一部入金・事務局照合まで握ると手戻りが減る（平田美知子さん回。分析 20260403_平田美知子_三浦美幸_分析.md Version 1.0）。

  • 不成約・難件でもふりかえりが具体的（前半のテンポ、経済ヒアリング、押しの強さ、理解の確認など）。チーム学習に向くメモが多い。五ヶ山本人がセールスした案件では酒井佑佳さん・石井和美さん・杉山敦子さん・出雲光さん・西村貴仁さん・吉岡拓也さん等（各 *_分析.md Version 2.0）。佐野・柴田・高田・吉田・森下・高橋の各回は濱野／和田担当で代理投稿だったため、下記「濱野りん」「和田めぐみ」を参照。

  • 長尺でも、成約後のツール案内や次回枠の調整まで会話を切らずに具体へ落とすと、初動の抜けが減る（野口由美子さん回・書き起こし末尾の Chatwork／LINE／スタートアップ日程の流れ）。

  • (2026-04-23 追記 / 出典: 20260416_永石陽子_五ヶ山昌代) 再アプローチ（辞退者本人からの再連絡）案件では、序盤に「経営年数・動ける強み」を事実で承認してから「感情先行で相手が置いてけぼりになる癖」を具体シーンで名指しする順序が刺さる。承認→癖名指し→近未来（本業トップセールス）と遠未来（自分の事業）を両方アカデミーで扱える旨の提示、という3段で決まるパターン。
  • (2026-04-23 追記 / 出典: 20260422_磯和なつ_五ヶ山昌代) 集中が切れやすい高齢顧客に対し、シナリオ核心ワードで真剣モードに引き戻す感知力は強い。引き戻せた瞬間に「今の一文でいちばん響いたのはどこ？」と言語化を取る一問を挟むと、金額提示時のブレが減る。

繰り返し論点・伸びしろ

  • 知識富裕層・長尺では「講師単独トークの連続分」を録画実測し、15分超えたら必ず相手へ質問を挟むルールにすると、石井和美さん回の分析と同じ改善が再現しやすい（20260304_石井和美_五ヶ山昌代_分析.md）。
  • 「褒め・共感」と「痛み・機会損失」の同時伝達が論点に上がりやすい。プライドが高い層ほど分析側からも指摘が集中（酒井さん回と集約分析など。3/13佐野・ピラティスは濱野担当）。
  • 慎重な相手に対して前半の説明が長くなりやすく、チャネル絞りを後から振り返るパターンは 濱野りん 欄（佐野・3/13）。
  • Zoom環境が悪い・長話型の相手での質問タイミングの難しさは 和田めぐみ 欄（吉田雅子・森下美枝子さん回）。
  • 他メンバーからの事前情報を先回りで打ち消そうとする動きと本音のズレの論点は 濱野りん 欄（柴田慶子さん回：澤田からの事前情報あり）。
  • 購入不可が先に出た回では、課題を3つに整理できる一方、本講座の「地図」を短く渡す尺が相対的に薄くなりやすい。タイムボックスと次回設計を早めに宣言すると講義化を防ぎやすい（大野真規子さん回・同上分析）。
  • (2026-04-23 追記 / 出典: 20260422_磯和なつ_五ヶ山昌代) 高齢・家族決裁案件では、本人の覚悟だけで即決を追うと「申し訳ございません」の自己評価に陥りやすい。即決しない案件でも「息子さんに今日なんて伝えますか？」「何がクリアになれば家族もOKと言いそうか」を本人の口から出すところまでがクロージング範囲。保留の質（期限＋家族トークの完成度）で評価する運用へ切替。
  • (2026-04-23 追記 / 出典: 20260416_永石陽子_五ヶ山昌代) 分割→一括切替を前提にクロージングを締める際、切替条件・期限を録画内で1フレーズ残す運用。事務局照合のブレ防止に効く。

繰り返し論点・伸びしろ

  • 同回の所感どおり、総括で相手の想いの言語化がもう一段はっきりするとクロージングが短くなるタイプ。語りが少ない見込み客ではラップアップ前の一文要約確認をテンプレ化するとよい。
  • シナリオ案件では台本が覚悟を代弁しやすい。福山回で効いた「期限付き数字＋行動承認」を、シナリオ直後の2問（刺さった一文／今日の決断の種類）とセットで再現する（宮崎みちこさん回・同上分析）。
  • (2026-04-23 追記 / 出典: 20260417_佐々木恵美_澤田亜美) 再ZOOM案件で「値段表提示前にAIシナリオ1・2を先に投入し、やるならプラチナを作ってから金額へ進む」順序を運用可能。明るく軽いノリの相手に対するシナリオ順序変更の型として再現価値が高い。苦笑い等の反応が出たら「いまのどこが苦笑いの部分？」と一問返し、本人の自覚を言語化させる運用とセットで効く。
  • (2026-04-23 追記 / 出典: 20260417_佐々木恵美_澤田亜美) 再ZOOMで録画が分断する場合、各本冒頭で「前回ここまで、今日ここから」の30秒再設計を入れる習慣を。再ZOOMは続きではなくリスタート。
  • (2026-04-23 追記 / 出典: 20260417_佐々木恵美_澤田亜美) プラチナ候補の相手には、序盤で「決済は一括・分割どちら派か」「クレジット枠の上限確認をしたことがあるか」の2問を挟み、クロージング段階での物理トラブル（カード枠不足・Wi-Fi・充電切れ）を事前に潰す運用を癖として定着させる。
  • (2026-05-26 追記 / 出典: 20260327_福山道子_澤田亜美) 自伝ブロック（自己紹介・経歴語り）が25分連続で発生。石井和美回の五ヶ山さんと同型。3分以内に収める台本を事前に用意し、タイマーで切る運用を推奨。
  • (2026-05-26 追記 / 出典: 20260327_福山道子_澤田亜美) 佐々木恵美回で追記済みの「序盤でカード枠確認」が本件でも実装されず、クロージング後に70分のロス（ライフカードJCB上限不明、PayPalエラー、ユニバーペイエラー）。再徹底が必要。

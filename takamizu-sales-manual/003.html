成功例（これは良かった）

実績数字で承認し、次の打ち手に接続する

  • 過去の売上ピークやローンチ実績を事実として置き、「だから次はSNS・オンライン」「だから仕組み化」と一本の線でつなぐ。
  • 実例: 市村睦美さん回（最高80万の事実からSNS・オンライン強化へ。分析 20260310_市村睦美_三浦美幸_分析.md Version 2.0）。吉岡理恵さん回（初ローンチ400万を承認し、次ステージの課題として忙しさ・組織化に接続。終盤クロージングは書き起こし欠落のため録画照合。分析 20260306_吉岡理恵_五ヶ山昌代_分析.md Version 2.0）。

当事者性を戻すクロージング

  • 「相談はしても、決めるのはあなた」「あなたでなければ商品化も教えもできない」と、依存ではなく決断の主体を本人に戻す。
  • 実例: 池田記代さん回（のりよさんが決める、美容院・健康・第二の柱を現場語彙で承認。分析 20260316_池田記代_三浦美幸_分析.md Version 2.0）。

悔しさ・使命を講座設計に直結させる

  • 不登校・栄養・発信実績など、ストーリーを「誰のどんな痛みを解く講座か」に翻訳し、次の行動がイメージできるまで落とす。栄養・現代食の不安が長く続く場面でも遮らず整理できると核が強い（シーン別は分析を参照）。
  • 実例: 森芙美子さん回（不登校時の悔しさ→講座化・毎日発信実績→方向を決めて集客、まで一文でつなぐ。分析 20260306_森芙美子_五ヶ山昌代_分析.md Version 2.0）。

多層の悩みを整理したうえで決め手を言語化する

  • 金銭・夫婦・集客・自己像が混ざっていても、最後に「あなただからできる役割」を短い言葉で贈る。
  • 実例: 金銭面・家族共有・リーダー像を整理し、遂行力と仕組み化で決め手のセリフに落とした回。

話し手が強い相手での主導権

  • 話好き・知識誇示が出やすい相手には、聞きすぎず方向転換を挟む。通期メモに「聞き過ぎない」と書いておくとチーム学習になる。
  • 実例: 野口由美子さん回、石井和美さん回（「何でも知っている」発言への対応として肯定しつつ自分で問題に気づく誘導）。

上部の褒めを封印し「できる貯金」で進める

  • 知識は多いが結果に自信がないタイプでは、褒めで逃がさず、タイミングを見て貯金（合意・小さなイエス）を積む。
  • 実例: 市村睦美さん回（三浦担当。分析 20260310_市村睦美_三浦美幸_分析.md Version 2.0）。

他コンサルと比較されても本気を言い切る

  • 比較が出たら、安易に貶さず「こちらも本気」と境界線をはっきり示す。
  • 実例: 野口由美子さん回（五ヶ山担当・通期メモ）。

成約後の初動も、決断の熱量が残っているうちに具体まで握る

  • 書面の受け取り、Chatwork や LINE の案内、スタートアップ用 Zoom の日程など、オンボーディングに近い一手を、別日に丸投げせず同じ面談の流れのなかで言語化すると、初動の抜けが減る。
  • 実例: 野口由美子さん回（五ヶ山担当。長尺セッション終盤でカード引き落とし説明・宿題・次月売上の打ち手まで同一画面で握る。書き起こしは字幕オフのため Whisper 取得。分析は 20260324_野口由美子_五ヶ山昌代_分析.md）。

期限付き数字と「すでに動いている事実」を重ねて承認する

  • 見込み客が掲げる月次・四半期の売上目標と、体験会参加・学習開始などの行動を同じブロックで承認すると、寡黙な相手でも決断の熱量が揃いやすい。
  • 実例: 福山道子さん回（澤田担当・6月まで60万／プラチナ12分割成約）。

体験会後の「次商品を言えなかった」を資産化し、マンツーへ接続する

  • 体験会だけでは次の一手が見えない見込み客には、「言えなかった」を責めず個別で橋を架ける。体験の感想から命題・時間・我慢のストーリーへ落とし、早期受講や本講座切替、月次の現実目標まで同じ面談で握ると初動が抜けにくい。
  • 実例: 柴田慶子さん回（濱野担当・プラチナ12分割成約。分析 20260304_柴田慶子_濱野りん_分析.md Version 2.0）。

一部入金で席を確保し、条件を口頭と事務局で一本化する

  • 返金不可・完済期限・「受講する前提」をその場で揃え、必要なら電話で事務局と整合を取る。振込経路が変わる場合もスクショ確認まで言語化する。
  • 実例: 平田美知子さん回（三浦担当・シルバー方向・一部入金と残額期限。分析 20260403_平田美知子_三浦美幸_分析.md Version 1.0）。

現場の数字で「天井」を一緒に計算し、導線の一点突破へ進む

  • 稼働日数×来店回数など、相手の言葉のまま売上上限を計算し、「このままでは届かない」を合意したうえで、公式LINE・ホットペッパー・インスタなど詰まり一箇所に矢印を返す。
  • 実例: 石井和美さん回（五ヶ山担当。分析 20260304_石井和美_五ヶ山昌代_分析.md Version 2.0）。

辞退者が本人から戻ってきたら、承認→癖名指し→未来設計の順序で締める

  • 2月の3チャレ等で一度辞退した顧客が本人の意思で再連絡してきた場合、覚悟はすでに一段上がっている。戻ってきた理由を本人の言葉で一度だけ確認し、深掘りしすぎない。そのうえで「経営年数・動ける強み」を事実ベースで承認し、「感情先行で相手が置いてけぼりになる癖」を具体シーンで名指し、近未来（本業トップセールス）と遠未来（自分の事業）を両方アカデミーで扱える旨を提示する3段で回収する。
  • (2026-04-23 追記 / 出典: 20260416_永石陽子_五ヶ山昌代) 再アプローチ本人連絡・シルバー12分割成約・カード枠上げ後に一括切替予定の事例。分割→一括切替の条件・期限は録画内で1フレーズ残す運用とセット。分析 20260416_永石陽子_五ヶ山昌代_分析.md Version 1.0。

再ZOOM案件では、値段表提示前にAIシナリオを先に投入する

  • 体験会参加後などの再ZOOMで、前回が値段提示前に中断した場合はリスタート扱い。冒頭30秒で「前回ここまで、今日ここから」を再設計してから、AIシナリオ1・2を先に当て、「やるならプラチナ」の結論を押し上げてからコース議論へ進む。
  • 明るく軽いノリの相手でも、苦笑いや沈黙が出た瞬間に「いまのどこが？」と一問返すと、本人の自覚が言語化されシナリオの効きが加速する。
  • (2026-04-23 追記 / 出典: 20260417_佐々木恵美_澤田亜美) 3/4体験会→再ZOOM・プラチナ方向作り・翌日カード会社確認後決済待ちの事例。分析 20260417_佐々木恵美_澤田亜美_分析.md Version 1.0。

「知られなければ申し込まれない」を一枚で言語化する

  • 努力と認知のギャップを短く言い切れると、サロン・物販・講座化のいずれでも決め手が残りやすい。
  • 実例: 野口由美子さん回（五ヶ山・成約。分析 20260324_野口由美子_五ヶ山昌代_分析.md Version 2.0）。

改善例・注意点（次に活かす）

褒め・共感と「痛み・機会損失」のバランス

  • プライドが高い・勉強熱心な見込み客ほど、承認だけだと決断が遅れる。同じブロックで「やらないと何が損か」を一枚入れる。
  • 実例: 酒井佑佳さん回（肯定と痛みのバランスを誤り決断に至らずと反省。分析 20260310_酒井佑佳_五ヶ山昌代_分析.md Version 2.0）。佐野恵子さん回（褒めと痛みを同時に伝えきれなかったと反省。五ヶ山担当 20260317_佐野恵子_五ヶ山昌代_分析.md と濱野担当 20260313_佐野恵子_濱野りん_分析.md をペアで参照）。

表記の一本化

  • 先頭の「▷結果」と所感の「成約／不成約」が食い違うと、分析も実績も壊れる。投稿前にワンクッション確認。
  • 実例: 吉岡理恵さん回（先頭不成約・所感成約の食い違い。CRM確定前に担当で一本化）。

📋期限付き保留の次アクション

  • 「3月末まで」「○日に返答」まで書けているのは良い。カレンダーと紐づくフォロー文テンプレを用意すると漏れが減る。
  • 実例: 石井和美さん、柏木肇さん、朝月希光さん。

他塾・ネットワーク優先の見込み客

  • 「今はあちらの課題が先」となったら、いつ再提案するか・何を達成したら再会するかを一文入れておく。
  • 実例: 杉山敦子さん回（ネットワークの学び期限優先）。佐野恵子さん回（複数塾を使いこなしてから次のステップ）。

「あとで数字が出てから決めたい」型

  • 体験会やローンチ結果待ちで今日の決断を先延ばしにする場合、何をもって「決断のタイミング」とみなすかを対話で握ると次回に繋がる。
  • 実例: 吉岡拓也さん回（3ヶ月後の体験会結果待ちで不成約。分析 20260313_吉岡拓也_五ヶ山昌代_分析.md Version 2.0）。

代表・会社としての検討

  • 個人の納得だけで終わらず、稟議・会社判断の期限と次コンタクトを明文化する。
  • 実例: 柏木肇さん回（3月末返答・会社として受講）。

第三者（占い等）に委ねる決断スタイル

  • 相手の価値観を否定せずつつ、ビジネス判断と占いの役割を分ける質問を用意する。
  • 実例: 朝月希光さん回（算命学の方に聞いてから、で3月末返答）。

難易度が高い見込み客（多方向・ノーハー慣れ・超長尺）

  • 一貫性や生活基盤の観点でビジネスが難しいと判断した場合も、事実ベースで記録に残す。無理な成約よりチームの安全線。
  • 学習歴が長く論点が散らばりやすい相手では、前半で「今日の終了イメージ」と「詰まり1つ」の合意を入れずに講義が膨らむと、終盤が「やりたいが今回断る」で止まりやすい。自伝・事例の連続尺は石井・柴田分析の上限ルールを参照。
  • 実例: 出雲光さん回（担当所感で難易度高。書き起こし確定後の分析は 20260310_出雲光_五ヶ山昌代_分析.md Version 2.0）。

⚠️三浦担当・通期の改善メモ

  • 池田記代さん回: 「もっとカットできるところがあった」→次回は無駄な往復を減らす編集意識。

語りが少ない相手のラップアップ

  • 総括で「想いの言語化」が一段足りないとクロージングが伸びる。締めの前に、相手の言葉で目的を一文要約して合意を取ると決断が早まることがある。
  • 実例: 福山道子さん回（澤田担当・所感）。

知識富裕層・長尺での「講師単独トークの連続分」

  • 教材でも、購買意欲は「お客様の未来」が主語である。実績・自伝が続くとワーキングメモリを占有し、比較・検討モードに入りやすい。
  • 実例: 石井和美さん回（五ヶ山担当・分析 Version 2.0）。対策の型は「1テーマ2〜3分で切る」「区切りごとに相手業態へ矢印を返す」。

事前情報と本音のズレ（冒頭で検証）

  • 他スタッフからの事前情報があるとき、潰しに行く前に一度仮説検証する。ズレると信頼を削る。
  • 実例: 柴田慶子さん回（濱野担当・澤田さんからの情報。分析 Version 2.0）。

理解速度と情報量のミスマッチ（善意が「強引」に聞こえる）

  • 説明量が相手の処理速度を超えると熱量が逆に不信になる。30分ごとに「いまの理解を一言で」復唱させ、ズレたら説明を止めて質問に戻す。3チャレ等フロント経由では冒頭10分で相手商品（誰に・何を・いくら）を固めてから本講座の地図に載せる。
  • 実例: 高田里恵子さん回（和田担当・3チャレ・不成約。分析 20260314_高田里恵子_和田めぐみ_分析.md Version 2.0）。

「聞くだけ」「借金」「あとで自分で」の三重防衛

  • 説明を厚くするほど負荷が上がる。投資不可が出たらオファーを短く切り、無料のミニ宿題を一つと次接触日だけ残す。「日付のない自分でやる」は次回設計で潰す。
  • 実例: 西村貴仁さん回（五ヶ山・2回目個別・不成約。分析 20260326_西村貴仁_五ヶ山昌代_分析.md Version 2.0）。

並行申込・別ルートあり（長尺でも優先順位が後出しになる）

  • 開始30分以内に「他に動いている申込・期限」を棚卸しし、今日の個別で決めることを一文で合意してから商品説明へ。終盤の「悩むね」以降で初めて別コミットが表に出ると、オファーの熱量が「せっかく」扱いになりやすい。
  • 実例: 吉田雅子さん回（和田・別プロデュース申込と現金の手元なし。分析 20260313_吉田雅子_和田めぐみ_分析.md Version 2.0）。

個別の位置づけの勘違いを早めに訂正する

  • 「説明会だと思っていた」等が出たら、今日のゴールと個別の役割をその場で揃える。
  • 実例: 佐野恵子さん回（濱野・3/13・不成約。書き起こし ## 15:00 前後の訂正シーン。分析 20260313_佐野恵子_濱野りん_分析.md Version 2.0）。

コンサル必須感×シルバー提示でも支払インフラが止まる

  • トーク以前に決済可否を切り分けて記録し、次回は「コンサル相当を最小単位で見せる」設計へ。不成約要因をCRMに残す。

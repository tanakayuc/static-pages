議論の流れ・背景

今回のテーマ ── 新エバーグリーンファネルを「後ろから」全チェック

田中が冒頭でテーマを提示。新しいファネルを具体的に作り込み、エバーグリーンのROAS（広告費用対効果）を一気に改善することにチャレンジしたいという方針。

進め方として「後ろから作っていく」考え方を共有。順番は、セールススライド → 体験セミナー（体験セミナーページ） → オープンチャットの運用メッセージ → LP、という並び。この一連を今回のMTGで片っ端からチェックしていく、という宣言からスタートした。

前提確認 ── オプトイン後の導線は「オープンチャットファースト」

田中が確認事項として、浜さん（協力者）の登録がまだかどうかを確認。雲野が「登録済みで、最初の3日間はセミナー誘導、その後はオープンチャットで2〜3日に一度YouTubeのURLが送られ、たまにセミナー誘導が入る」という現状の流れを説明した。

田中は自分がLINEを運用している経験から、「朝担当・夜担当」で役割分担する発想を紹介（自身と片岡さんの例：朝10時に片岡さん、夜22時に田中で、ミーティングなしでも一人語りで機能する）。午前はノウハウ（動画を送る）、午後は体験セミナーへの誘導、という時間帯別の目的分けが良いとの考え。

その上で「LINEファーストで行くか、オプチャ（オープンチャット）ファーストで行くか」という論点を提示。雲野は「オープンチャットファーストの方がいいかなという気がする」と回答。田中も「運用コストはかかるが、リアルタイムで配信していく方が強いと思っていた」と同意し、LINEはコストがかかる分、特典受け取りやアンケートなど限定用途でオープンチャットからLINEへ流す使い方に絞る方針でまとまった。

つまり、サンキューページはオープンチャットへのシンプルな誘導ページにし、動画はオプトインLPの方に置く（サンクスページには動画をつけない）という切り分け。オープンチャットから体験セミナーへ、午前はコンテンツ・午後は体験セミナーのプッシュ、という流れで進める方向。

体験セミナーの司会 ── プロの司会を軸にした運営体制へ

田中は「体験セミナーページのクオリティが、バックエンド開設につながる体験セミナーの成否を左右する」とし、最初の何回かは中田本人が登壇し、その後は録画配信でもよいのではという考えを示した。

中田から、実際にはすでに「視界」（=司会）を担うプロの司会者がいるという情報共有。プロのアナウンサー経験者を中心に体験セミナーを回し、その人に「歯科医とは何ぞや」的な教育部分も含めて三人ぐらいの体制で進めてもらう構想があるとのこと。

田中は「それでいい。逆に運動さん（雲野）にはあかまさんのセミナーを見てほしい。放送してるだけか司会が入っているのか、司会を入れる方針でいいと思う」とコメントし、司会を軸にする方向を後押しした。

セールススライドの構成 ── セミナーと個別説明会の境界が曖昧

現行スライド資料を画面共有しながら精査。雲野が「上と下は同じスライド」「これは個別のセールス用スライド」と説明する場面があり、田中は「セミナー用と個別用が混在していて、どっちやねんという状態になっている」と指摘。

田中が特に問題視したのは、スライド中の「どんな感情を感じてますか？」という直接的な質問。中田も「ここは本来セミナースライドなしで進めるべき部分。ヒアリングで聞き出す領域で、見せるところではない」と同意した。

田中の整理：本来はヒアリングに集中し、お客様自身の口から「原因」「未来像」が明確になった段階で「では私たちのご提案を」という流れが理想。現状のスライドは主観的な決めつけが強く、真の原因が曖昧なまま進んでいる印象があるとの指摘。

コアメッセージの再確認 ── 「ビリーフノート」を軸にした脳→行動の変化ロジック

田中は、以前もっとロジックが明確に語られていた記憶があると述べ、下記のトーク構成をその場で言語化して提示した。

  1. 脳が散らかっている状態（脳疲労）が根本原因で、うまくいかないことのオンパレードが起きている
  2. 脳の状態を変えられる唯一の手段が「ビリーフノート」
  3. ビリーフノートが変わることで脳が変わり、お片付けができるようになる（＝一秒収納アカデミー）
  4. さらにその先で、片付けができるだけでなく、自分に自信を持ち「本当の自分」に戻り、自分のやりたいことに向き合える「強くて優しい女性」＝「ナチュラルウーマン」になっていく

田中は「そこがちょっとぐにょぐにょしている印象。もう少し整理してほしい」と述べ、セミナーで落とし込み → 個別説明会で確認、という基本構造に対して、スライド自体が「全部個別向け」の作りになっている点を問題視した。

オープニング5〜10分の重要性 ── 「他と違う」権威性とワクワク感の刷り込み

田中は「オープニングの最初の5〜10分が最大級の鍵」とし、現状のスライドは「構成をはめているだけ」で、権威性・独自性・ワクワク感を作り込む設計になっていないと指摘。

自身の経験として、自分のYouTube1本目のオープニングを、当初2時間撮影 → 自分編集で10分 → タクトさんが5分に凝縮 → その後も十数回書き換えを重ねてブラッシュアップした経緯を共有し、「そのレベルで作った方がいい」と提案した。

中田は、実在するプロの司会者（アナウンサー経験者、りえさん）のイメージを共有。「皆さんこんにちは」という明るく丁寧な導入から始まり、用意された台本を演技も含めてしっかり話してくれる、という体制を想定していると説明。田中は「もう司会がどこまで話すかの設計次第」とコメント。

司会と中田本人、それぞれの役割分担の論点

中田から「私は毎回いるわけではない」という前提が共有され、司会が「こんにちは」と繋いだ後、中田の録画動画を流す形になる想定であることが確認された。

田中は、オープニングで確立したい感情として「常識の破壊」「権威性の確立」「ワクワク感の醸成」の3点を挙げ、これは一言一句作り込むべきだと主張。中田も「私もそう思う」と同意。

続いて、中田の話す内容を動画で流す場合の「生感」の欠如について中田自身が懸念を提示（「私の言葉である優位性はあるが、場の空気を掴むことが動画でできるか」）。田中は「どちらもできそうだから」とコメントし、中田が「両方やって数字を見て比較したらいい」と提案。田中も「一般的にナビゲーターにはあまり期待しないタイプ。ルール説明の後、りょうこさんが一番厳しい話を最初にする形で入るのがいいのでは」と自身の好みを述べた。

最終的に中田が「両方のいいとこ取り」として整理：司会は感じよく短時間でしっかり話す（一語一句用意されたものを話す）→ 短時間で中田の動画へ → 中田がファッション（情熱）を込めて「ただのお片付け塾ではなく、しっかりした学校である」ことが伝わる話をする、という役割分担でまとまった。

中田は、リアルタイム進行における司会の重要性（参加者がバラバラ入ってくる状況をうまく取り仕切れるかどうかで信頼感が変わる）にも言及。田中は、完全自動化（録画のみを流す・人を介在させない）チームの事例も紹介しつつ、「途中参加の遅刻許容ラインなど、動画のみだとコントロールが難しい」との考えを共有した。

スライド・トークのボリューム設計 ── YouTube台本レベルの作り込みを

田中は、ナビゲーターと中田それぞれのトークを「YouTube台本のノリ」でボリューミーに作るべきだとし、現状のスライドは「薄すぎる」とコメント。

中田から「流れ自体がこれで合っているのか、大枠から違うのかを先に確認したい」という懸念が出され、田中は「大きな視点から勝負しに行きたい」と回答。

「片付けのあり方」を問い直す大きな視点でのオープニング設計

田中が、これまでの「私が片付けられなかった、だから大丈夫」という個人の物語軸ではなく、「片付けというもの自体のあり方を宇宙的に問い直す」大きな視点からのオープニングを提案。

「今の日本人が抱えている問題は何か」「AI時代になぜ私たちは幸せを感じられないのか」「他人に振り回される、自分軸がない」といった社会的な問題提起から入り、「本当の自分を映す鏡が存在していない。それを唯一作れるのがお片付けだ」という論理展開を提示。中田はおおむね同意した。

さらに、「なぜ一時的に良くなってもまた元に戻るのか」という問いに対し、中田が独自の回答を展開。「みんな部屋を綺麗にする（空間的な話）としか考えていないが、片付けは時間的なもの。綺麗な状態を維持するための行動様式（ルール）が必要」という説明。具体例として「洗い物を3つ貯めたら洗う、というようにルール化するとシンクは常に綺麗」という話を紹介。中田はこの「一貫性と継続性」の欠如が根本原因だとまとめた。

田中は、この話の持っていき方自体はいいとしつつ、「決め技っぽさが足りない」とコメント。雲野が言っていた「脳の癖」というキーワードとどう結びつけるかを問い、中田は「脳疲労の話。部屋が綺麗でも脳がノーヒーロー（余力がない）状態だとやる気が上がらず、計画もできない」と回答した。

起業のロジックとの類推 ── 思考→行動→結果、抽象度を上げた説明

田中は「これはさっき思ったが、起業の話と一緒。思考があって、行動があって、結果があるという構造で、脳の機能面から一度抽象度を上げて説明した方がいい」と提案。

多くの人は行動を変えようとするが、実は整理されていない思考・脳の癖が根本原因になっている。それを放置すると家族関係の悪化や、子どもの引きこもり傾向、勉強しなくなる等、家の状態が人生全体に影響していく、という深掘りロジックを展開。

「思考の状態が整理できていないのは、脳が疲労状態にあるから」というつながりで、唯一その状態をひとつずつ解きほぐせるのが「ルール」であり、それを「ビリーフノート」という名前でメソッド化している、という核心的な位置づけを改めて言語化した。

独自性の出し方 ── 「周りから固めていく」トークの型

田中は自身のトークの特徴として、「一本強く言い切るための段取りを、周りから構造的に固めていくやり方をしている」と説明。「確かにそれしかない」という状態に持っていくことで、独自性・説得力が生まれるという考え。

「他のお片付け系では、自分ができたことをそのまま伝えているだけかもしれないが、私たちは何千人もの脳の動きを研究し続けて、ADHD気質の人も変われる仕組みを研究した結果たどり着いた唯一の結論であり、独自のメソッドだ」という語り口を提案。中田は「本当にそう」と強く同意した。

個別相談のオファー設計 ── 「餌」の欠如と誘導率3%の問題

田中が個別相談（旧・個別説明会）のオファー設計について質問。現状は「個別説明会」だけをシンプルに押している状態で、誘導率が3%程度になっているとのこと。中田は「餌がない」ことに同意した。

田中は他社事例として、ホットペッパー系のサロンリピート施策で使われる「診断会」オファーや、ピングアイマンさん（協力者）の事例（診断がついているのに成約率40%程度が出ている）を紹介。誘導率と成約率の関係は「誘導率を上げると成約率が下がるパターンもあれば、両方上がるパターンもある」という前提を共有した。

中田から、雲野が作成した「同タイプ診断」（AI活用）がすでにあるという情報共有。ただし、中身が曖昧なまま個別相談に誘導すると、セールス担当が診断結果をうまく扱えず、意味のないやり取りになってしまう懸念を中田が指摘。

田中の提案：一旦はオファーをつけず、「金額を知りたい方・カリキュラムの詳細を知りたい方は個別説明会へ」というシンプルな訴求に変更する。名称も「アカデミー個別説明会」とし、一秒収納アカデミーで何をやるか・どんなカリキュラムかを知りたい人向け、という位置づけに整理した。

中田は「同タイプ診断」の名称についても、「本音を引き出す」という表現を提案したが、田中が「本音は引き出したくないのでは」と疑問を呈し、雲野も「本音がわからないことすら自覚していない人が多く、ピンとこない可能性がある。もう少し手前の訴求の方がいい」と補足。最終的には、グローバルキッズマム案件の個別説明会ページ（3ステップ構成：診断 → 成長方針の相談 → アカデミーの案内）を参考事例として共有し、中田も「これはいい。ステップが明確なのがいい」と好感触を示した。

オプトインページ・動画選定の論点

田中は、オプトインLP案が「ほわわっとしている」と指摘し、「どの動画を使うか」が鍵になると提起。現在採用している動画について、選定理由（視聴維持率がいいのか、再生数が多いのか、内容が合っているのか）を確認したところ、雲野は「内容がADHD味・自己肯定感の低さがターゲットに合っている」と回答。ただし田中はその動画タイトル・訴求（「片付けの本質は片付けの問題じゃない」等）が「何が言いたいのか分かりにくい」とコメントし、別の動画（「知らない人が始める理由」）の方がシンプルでいいのではと提案した。

LPの構成については「YouTubeのタイトルをそのまま大きく分かりやすく載せた方が引きが強い」との考え。オプトイン後はオープンチャットへ誘導し、朝はYouTube、夜は体験セミナー、というリズムで運用する方向。

「黒歴史シリーズ」動画をYouTube起点に統合

雲野から、中田の半生をシリーズ化した「黒歴史動画」（小学生編〜結婚〜再婚〜出産〜娘の小学校入学まで、各2〜3分）が既にあり、現状は個別相談申込者向けに17日間連続配信していることが共有された。

田中は「このファネルの弱点は、順番の制御ができないこと」と指摘。黒歴史シリーズを見せたい場合、リンクを1本に絞り、例えば「10日に1回」のペースで流す設計が必要と提案。中田は「シリーズとしては順番があるが、途中から見ても問題ない内容（中学生編から見ても違和感がない）なので、順番は気にしなくていいのでは」と回答。

田中は、ステップメール（全員同じ導線）と、LINEオープンチャット（都度その時の鮮度のコンテンツを出す場）とで役割を分けるべきとの整理を提示。ステップメールでは体験セミナーへのプッシュを継続しつつ、オープンチャットでは黒歴史シリーズ等をぐるぐる回していく設計。

裏テーマとして「YouTubeアカウントを広告の力で強化する」狙いがあるとし、黒歴史動画・お絵かきムービー（現在は別サービス「宴」に掲載）も含め、すべてYouTubeに寄せてプレイリスト化し、どこかで引っかかった視聴者が他の動画にも遡れる導線を作りたいという方針を共有。中田も同意し、「中田りょうこ 黒歴史 プレイリスト」としてYouTube上にまとめる方向で一致した。

田中は、体験セミナー中に司会からチャンネル登録を呼びかける施策や、登録のスクリーンショットを送った人へのプレゼント施策のアイデアも出した。

考え方・マインドセット

「後ろから作る」設計思想

セールススライド → 体験セミナー → オープンチャット運用 → LP、という「後ろから」順にファネル全体を作り込む。バックエンドの完成度が弱いまま手前を作っても意味がないという田中の一貫した考え方。

オープニングの5〜10分がファネル全体の生命線

体験セミナーの冒頭で「常識の破壊」「権威性の確立」「ワクワク感の醸成」を一言一句作り込むことが最重要という位置づけ。田中は自身のYouTube動画オープニングを十数回ブラッシュアップした経験を引き合いに、同水準の作り込みを求めた。

「周りから固めていく」トークの型

一本の強い結論に持っていくために、いきなり主張するのではなく、構造的に周辺から埋めていき「確かにそれしかない」という状態に相手を導く、という田中のトーク構築の型。

セミナー用スライドと個別用スライドは役割が違う

セミナーは「落とし込み」、個別説明会は「確認」という役割分担。現状スライドがこの境界が曖昧になっている点が課題として共有された。

ステップメールとオープンチャットの役割分担

全員に同じ内容を届けたい情報（体験セミナー誘導等）はステップメールで、その時々の鮮度のあるコンテンツ（黒歴史シリーズ等）はオープンチャットで回す、という設計思想。

ノウハウ・気づき

LINEファースト vs オープンチャットファースト

運用コストはオープンチャットの方がかかるが、リアルタイム配信の強さを重視し、今回はオープンチャットファーストを採用。LINEは特典受け取り・アンケートなど限定用途に絞る。

個別相談のオファーは「診断→方針相談→案内」の3ステップ構成が効果的

グローバルキッズマム案件の個別説明会ページを参考事例として共有。ステップが明確に分かれていることが訴求力を高めるポイント。

動画のプレイリスト化でYouTubeアカウントを強化

黒歴史シリーズやお絵かきムービーなど、他サービスに散らばっているコンテンツをYouTubeに統合し、視聴者が他の動画にも回遊できる導線を作る。広告強化の裏テーマも兼ねる。

誘導率と成約率はトレードオフとは限らない

誘導率を上げると成約率が下がるケースもあれば、両方上がるケースもある。オファー設計次第で結果が変わるため、一律の正解はない。

決まったこと

  • ファネル全体を「セールススライド → 体験セミナー → オープンチャット運用 → LP」の順で後ろから見直す

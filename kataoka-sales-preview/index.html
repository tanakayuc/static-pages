なぜ「Claude Code」なのか

ChatGPT、Gemini、Claude…AIツールはいろいろありますが、本講座ではClaude Codeに特化しています。

その理由は、「アドバイスをくれるAI」と「実際に手を動かしてくれるAI」の違いにあります。

ブラウザ上のAI（ChatGPT、Claudeなど）

「お家を掃除したい」と聞くと、「こうやって掃除するといいですよ」とアドバイスはくれる。

でも、実際に掃除してくれるわけではない。

Claude Code

「こう掃除しますね」と提案するだけでなく、必要なツールを用意し、次回のマニュアルも作りながら、実際にお家の中をガシガシ掃除してくれる。

つまり、Claude Codeはあなたの「手足」になってくれるAIです。

  • パソコンの中のPDFを読み込んで請求書をまとめる
  • Excelのデータを整理して報告書を作る
  • 毎月の定型作業を自動でやってくれる仕組みを作る

指示を出すだけではなく、実際の作業を代わりにやってくれる。

だからこそ、業務改善や仕組み化、「今、人間がやっている作業」の代替が可能になる。

経営者にとって、これは大きなインパクトがあります。

OUR VALUES

このコミュニティで大事にしていること

地域枠を絞っている理由

本講座は、各都道府県3名までの地域枠制を取っています。

理由はシンプルで、コミュニティのメンバー同士でお客様を奪い合うようなことが起きないようにするためです。

同じエリアで競合する関係になってしまうと、「成功事例の共有」も「失敗の相談」も気軽にできなくなってしまう。

だからこそ、地域を区切って、全員が安心して情報交換できる場を作ります。

全員で勝つ

1人だけ勝っても意味がない。仲間と一緒に成功する。

成功事例も失敗事例も、すべて包み隠さず共有し合う。

別に、同じ地域の人と無理に協力する必要はありません。ですが、ある人の成功事例が、別の地域の人のヒントになることはたくさんあります。ある人の失敗が、別の地域の人の回避策になることも、たくさんあります。

それが、コミュニティで学ぶ最大の価値です。

全てはテストである

最初から完璧を目指さない。やってみて、改善する。

失敗は「データ」。次に活かせばいい。

人に貢献したいという思い

稼ぐことだけが目的の人は難しい。

クライアントの課題を解決したい。地元の経営者を助けたい。

この気持ちがないと続かない。

こういう方をお迎えしたい

「自分一人だけ稼げればいい」ではなく、

実践を通して、コミュニティを一緒に盛り上げていきたいという気持ちを持つ方。

成功事例も失敗事例も共有しながら、仲間と一緒に成長していける方。

こういう方は、お断りします

反対に、こういう方は申し訳ありませんがお断りします。

  • 楽して稼ぎたい
  • 自分さえ得すればいい
  • 成功事例も失敗事例も共有したくない
  • 他のメンバーから情報だけ取って自分は出さない
  • すぐに結果が出ないと諦める

私たちの理念とは合いませんので、お互いのために、参加はご遠慮ください。これは、すでに参加している仲間を守るための仕組みです。

「二段階審査制」で、お互いに本当に一緒にやっていけるかを確認します。

なぜ二段階審査制なのか

お互いに「この人と一緒にやっていきたい」と思えることが大切だからです。

私からあなたを審査するだけでなく、あなたからも私を審査してください。違うなと思ったら、辞退して構いません。

そして、利己的な方や、コミュニティを盛り上げる気持ちがない方とは、こちらからもお断りさせていただきます。これは、すでに参加している仲間を守るための仕組みです。

お申し込みの流れ（5ステップ）

下記ボタンから直接「第一次審査フォーム」にお進みいただけます。事前に相談したい方は、ご相談用LINEからお気軽にメッセージをお送りください。

1

第一次審査フォームにアクセス

このページの「第一次審査に申し込む」ボタンから審査フォームに進めます。事前に相談したい方は、ご相談用LINEからお気軽にご連絡ください。

2

第一次審査フォームに記入

フォームに必要事項をご記入ください。

第一次審査フォーム記入項目:

  • お名前
  • お住まいの都道府県
  • 現在のお仕事
  • なぜ今回申し込もうと思ったのか（具体的に）
  • 本日のセミナーで響いたポイント
  • 講座を通して実現したいこと
  • 面談希望日時
第一次審査でお断りすることがあります。熱意が低い、内容が抽象的すぎる場合は、面談に進むことなくお断りすることがあります。「ちゃんと考えて書いてくれた方」だけ、次のステップにお進みいただきます。
3

第一次審査結果のご連絡

事務局より第一次審査の結果をご連絡します。通過された方のみ、面談日程の調整に進みます。

4

第二次審査（15分面談）

オンラインで15分間の面談を行います。

  • 第一次審査の内容を確認
  • お互いの相性を確かめる
  • 地域枠の最終調整
第二次審査でもお断りすることがあります。ここで「現段階ではご案内が難しい」と判断した場合は、決済フォームをご案内しないこともあります。あらかじめご了承ください。
5

決済・本申し込み

両者合意の場合、お支払い方法をご案内します。
入金確認後、「Chatworkグループ」をはじめとするコミュニティへ正式にご参加いただけます。

FAQ

よくある質問

AI未経験でも大丈夫ですか？

はい、大丈夫です。講座はAI未経験を前提に設計しています。「見せ方」と「順番」のテンプレートがあるので、学ぶ気持ちと実践を重ねることで成果につながりやすい設計です。

本業（士業・コンサル業）と両立できますか？

本講座はむしろ、すでに本業を持っている方にこそ最適化されています。本業の片手間で取り組めるカリキュラム設計です。AI導入支援を入口に、本業（融資・財務・補助金・労務など）の案件まで連動して増えるよう設計しています。月1セミナー＋月2グループコンサル＋月1ロープレなので、週末や夜の時間で十分に参加可能です。

既存クライアントへの提案で失敗したら、関係が崩れないか心配です

提案の型と順番が決まっているので、関係性を壊さずに自然にアップセル化できる構造です。スライド・トークスクリプト・反論切り返し集まで全てお渡しします。

地方に住んでいますが問題ないですか？

むしろ地方のほうがチャンスです。ネットの全国大会ではなく、地方の大会で戦えます。競合が少なく、チャンスが大きい状態で始められます。

年齢が高めですが、本当にできますか？

むしろ経験豊富な方のほうが有利です。経営者からの信頼を得やすく、落ち着いた印象で安心感を与えられます。重ねた人生経験が最大の武器になります。

しゃべり下手なので不安です

セールスは必ずしも「しゃべり上手」だけが成功するわけではありません。「しゃべり下手」な方がお客様に信頼されやすい、ということもあります。台本とロールプレイングで準備しますので安心してください。

家族に相談してから決めたいのですが…

正直にお答えします。本講座は「即断即決できる方」に来ていただきたいと考えています。

家族への相談自体を否定はしませんが、「自分の意思で動けない方」よりも、「自分で決めて動ける方」の方が、AIコンサルとしての成果は出やすいです。

体験セミナー参加から3日以内（72時間）が申し込み期限となります。「家族に相談する時間がほしい」という方は、ぜひセミナー終了後すぐにご相談を始めてください。

AI領域は未経験で、踏み出すのが怖いです

「AI未経験」を前提に設計しているので、ゼロからでも問題ありません。むしろAI未経験の状態で、対面営業の経験と社長との信頼関係を持っているのは強みです。AI導入支援は、社長の懐に入る最強の入口になります。

補助金の知識がまったくありませんが、提案できますか？

はい、できます。本講座では補助金支援総額5億円超の園田芽衣氏が監修する「独自開発の補助金支援ツール」を使えば、業種・地域に合った補助金のリサーチから、事業計画書の策定までAIがサポートします。

さらに、「これってどう書けばいい？」「うちのクライアントは対象？」など、判断に迷う場面は、園田氏の補助金専門家チームへChatworkで直接相談できます。

「自分は補助金の素人」と思っている方でも、専門家のバックアップを受けながら、クライアントに堂々と補助金提案ができるようになります。

# 1秒収納 セールスマニュアル

このマニュアルは、個別セールスの分析から抽出した成功パターン・改善ポイントを蓄積し、セールス品質を向上させるためのものです。

分析件数が増えるたびに更新する。

---

## 心構え: わたしたちは人生の救助隊である

セールスは「売り込み」ではない。片付けに悩む人を救う「救助活動」である。

あなたの劇場にお客様がワクワクして来てくれている。楽しませる心意気で迎えよう。

## セールスの基本フロー

```
1. 開始・環境確認（5分）
      ↓
2. ヒアリング（20〜30分）
      ↓
3. 講座説明（20〜30分）
      ↓
4. 価格提示・クロージング（15〜20分）
      ↓
5. 契約手続き or 次回アポ設定
```

## STEP 1: 開始・環境確認

### 笑顔で迎える（最重要）

Zoomで相手が入ってくるタイミングで、とびっきりの笑顔で迎える。

お客さんはあなたの劇場に来ている。楽しませる心意気で。

その笑顔を動画に残すつもりで。

### 必ず確認すること

開始時に以下を確認する。これを怠ると、途中で家族の介入などが発生し、集中が途切れる。

確認トーク例:
```
本日はお時間いただきありがとうございます。
今日は大体60分〜90分ほどお時間いただく予定ですが、
この後のご予定は大丈夫ですか?

また、今いらっしゃる場所は、周りにご家族とかいらっしゃいますか?
もし途中で呼ばれそうでしたら、先にお伝えいただければ
中断しても大丈夫ですので。
```

## STEP 2: ヒアリング

### 引き出すべき情報

| カテゴリ | 質問例 | 目的 |
|----------|--------|------|
| 流入経路 | 「どんな経緯でセミナーにご参加されたんですか?」 | 広告効果の確認 |
| 片付けの悩み | 「具体的にどんなところが気になっていますか?」 | 表面的な悩みを把握 |
| 生活状況 | 「お仕事は?ご家族は?」 | 時間・環境の制約を把握 |
| 本当の悩み | 「片付けられないことで、どんな気持ちになりますか?」 | 深層の悩みを引き出す |
| 理想の未来 | 「片付いたら、どんな生活になりたいですか?」 | 講座で得られる価値と紐づける |
| 他の講座 | 「他にお片付け講座や本で学んだことはありますか? それはいくらでしたか?」 | 比較対象・アンカリングに使う |
| 検討中の講座 | 「他に検討しているお片付け講座はありますか?」 | 競合との差別化ポイントを伝える |
| 家計の決定者 | 「家計を握っているのはどなたですか?」 | クロージング時の対応を準備 |

### 損している金額を引き出す（SPIN営業法）

片付け苦手で損していることと金額を具体的に引き出す。価格提示時のアンカリングに使う。

質問例:
```
「片付け苦手で、損してるなあと感じることってありますか?」

「例えば、外食が増えちゃうとか」
「買い物が多くなるとか」
「同じもの買っちゃうとか」

「月にどれくらいですか?」
「年間にすると...結構な金額ですよね」
```

引き出すべき損失:
- 外食費（家が散らかっていて料理する気にならない）
- ダブり買い、衝動買い
- 服を買ったけどタグついたまま
- 探し物の時間（年間60時間以上）
- 家の何割が活かせていないか（家賃・住宅ローンで割合を出す）

### 深掘りのコツ

顧客が話し始めたら、遮らずに聞く。相槌を打ちながら、以下のフレーズで深掘りする。

```
「それは大変でしたね」（共感）
「もう少し詳しく教えていただけますか?」（深掘り）
「それって、どんな気持ちになりますか?」（感情を引き出す）
「いつ頃からそう感じていらっしゃいますか?」（時間軸を確認）
```

### 成功事例: 深いヒアリングができたケース

桜庭はるみさま（2026年4月）:
- 片付けの悩み → 同居のストレス → 自己肯定感の低下 → 離婚も考えた、まで引き出せた
- 「私が自分を大切にしてあげないと私がかわいそう」という本音を引き出した
- この本音をクロージングで活用できた

## STEP 3: 講座説明

### 説明する内容（チェックリスト）

- [ ] 1秒収納の理念（一生片付けに困らない人に成長する）
- [ ] 講師の自己紹介・体験談
- [ ] 氷山の図（顕在意識5%、潜在意識95%）
- [ ] ビリーフノートの書き換え
- [ ] 片付けの3つのポイント
- [ ] 常識の破壊（パラダイムシフト）
- [ ] 受講生の声（ビデオ）
- [ ] サポート内容12個
- [ ] 感謝祭・ルームツアー
- [ ] 一生サポート宣言

### 常識の破壊（パラダイムシフト）

顧客の「当たり前」を壊すことで、講座の価値を印象づける。

例:
```
「片付けって、どっちだと思いますか?
A『コツコツ毎日やる』
B『一気に集中してまとめてやる』」

（少し黙って考えさせる）

「実は答えはBなんです。
なぜならば、一気に集中してやることで脳が活性化して、
頭に残りやすいんですね。」
```

ポイント:
- 緩急をつける。考え込んだり、「うーむ」とか。少し黙ってみたり
- 「私、こんな気がするんですけど、合ってますか?」と顧客に考えさせる

### 価値の棚卸し（価格提示前に必ず行う）

価格を見せる前に、顧客自身に「自分の問題」と「得られる未来」を言語化させる。

棚卸しトーク例:
```
ここまでお話しさせていただきましたけど、
ちょっと一度整理させてください。

{顧客名}さんの今の状況をまとめると、
・{悩み1}
・{悩み2}
・{悩み3}

この理解で合っていますか?

で、{顧客名}さんが1秒収納を通じて手に入れたい未来って、
さっきおっしゃっていた「{顧客が言った理想}」
っていうことですよね?

もしそれが手に入ったら、{顧客名}さんの生活って
どう変わりそうですか?
```

## STEP 4: 価格提示・クロージング

### アンカリング（価格提示の前に）

価格を見せる前に、顧客が既に使っている・使ったことのある金額を引き出しておく。

飛びつくべきアンカー:
- 子供の大学の入学金
- 車の代金
- 家の価格（住宅ローン）
- 旅行（満足感）
- 英会話など他の講座
- 「これまでに高額講座を買ったことある?」「それ、いくらだったんですか?」

ヒアリングで引き出した「損している金額」も使う:
- 「さっき外食で月3万円とおっしゃってましたよね。年間36万円。5年で180万円ですね」
- 「探し物に年間60時間。時給1000円としても6万円。10年で60万円ですね」

### 価格提示

定価: 66万円（税込）
海外在住者: 60万円（消費税免除）
当日特別価格: 60万円（現金一括）

分割オプション:
- 6回払い: 頭金16.5万円 + 残額を6分割
- 12回払い: 頭金16.5万円 + 残額を12分割
- カード後払い分割: 24回、36回

### 「やりましょうよ！」とさわやかに言う

クロージングでは、さわやかに「やりましょうよ！」と言う。

引きすぎない。背中を押す。

### よくある反応と切り返し

| 反応 | 切り返しトーク |
|------|----------------|
| 「高いですね」 | 「そうですよね。では、{顧客名}さんにとって、片付けられない状態がこのまま続くことと、60万円を投資して一生片付けに困らない自分になること、どちらの方が価値がありますか?」 |
| 「考えます」 | 「何について考えたいですか? 金額ですか? 内容ですか? 具体的に教えていただければ、一緒に考えられるかもしれません」 |
| 「家族に相談します」 | 「どなたに相談されますか? ご主人ですか? もしよければ、ご主人にどう伝えるか、一緒に整理しましょうか」 |
| 「銀行に確認してから」 | 「今この場でカード会社に電話して確認することもできますよ。5分くらいで分かります。お電話されますか?」 |

### 「金額がムリです」「10万円くらいと思ってました」への対応

安い金額しか考えていなかった人への対応:

```
「いいですよ。ここまで頑張ってこられましたもんね」
「こちらがお願いして無理してやっていただくものでもないんで」
「あなたの考え方、意見を尊重します」
```

（いったん認める）

```
「ところで...1年後、あなたの望む状態になっていますかね?」
```

（その先の状態を想像していただく）

顧客が「いや、無理です」と言ったら、そこで再クロージングする。

### 背中を押すトーク

「私から進めません」ではなく、「背中を押させてください」のスタンスで。

トーク例:
```
{顧客名}さん、私からお伝えしたいことがあります。

さっき{顧客名}さん、「{顧客が言った本音}」
っておっしゃいましたよね。

私、あの言葉を聞いて、本当に今がタイミングだと思いました。

{顧客名}さんは、もう十分すぎるくらい頑張ってこられました。
ここから先は、1人で頑張るんじゃなくて、
私たちと一緒に進んでいきませんか?

背中を押させてください。
```

### ヒアリング情報の活用

ヒアリングで引き出した顧客の言葉を、クロージングで使う。

例:
```
{顧客名}さん、さっき「私が自分を大切にしてあげないと私がかわいそう」
っておっしゃいましたよね。

この講座に参加するって、まさに「自分を大切にする」行動だと思うんです。

今日、ここで決めるって、{顧客名}さんが{顧客名}さん自身に
「あなたは大切だよ」って言ってあげることだと思うんですね。

{顧客名}さん、ご自身のために決めませんか?
```

## STEP 5: 契約手続き

### 現金一括の場合

その場でネットバンキングのお手伝いをする。

```
「今、ネットバンキングでのお支払いのお手伝いしますよ」
「一緒の方が不安なくできますよね」
```

### 成約後のキャンセル防止（必須）

成約後、必ず以下の質問をする:

```
「なぜ、この講座を受けようと決心されましたか?」
```

（顧客に自分の言葉で理由を言わせる）

```
「成約後に『やっぱりやめたい』という感情が必ず湧いてきて
葛藤があると思いますが、その時どうしますか?」
```

（「大丈夫です」と言わせる）

```
「自分で少しやってみたら『できた!』ってなりますが、
それは長続きしそうですか?」

「夫が『手伝うよ』と言ってくれて、二人で片付けたらできた、
という場合も...これも長続きしますかね?」
```

## 保留の場合のルール

### LINEでのお断りはなし（約束）

保留になった場合、LINEでのお断りはなしと伝える。

理由:
```
「もう講座が始まっています。
ナチュラルウーマンとして、『約束を守る』ということに
チャレンジして欲しいのです。

ひとつひとつの小さな約束を守ることができるようになって、
片付け苦手を克服できる人に変化成長できるのです。」
```

## 成功パターン集

### パターン1: 自己開示で信頼を得る

講師自身の失敗談・苦労話を語ることで、「この人は分かってくれる」という信頼を構築する。

事例（桜庭はるみさま）:
- 講師が「私も食事の買い物ができなくなった」「IQが下がったと思った」と語った
- 顧客は「私もそうかも」と共感し、心を開いた

### パターン2: 強い承認で自己肯定感を上げる

自己肯定感が下がっている顧客には、強い承認を繰り返す。

フレーズ例:
- 「{顧客名}さんは1mmも悪くないです」
- 「よく踏ん張ってこられましたね」
- 「それは{顧客名}さんのせいじゃないですよ」

### パターン3: 顧客自身に「気づき」を言語化させる

顧客が自分で問題を整理できると、講座の価値を自然に感じる。

事例（古賀いづみさま）:
- 担当者が「荷物ってどんなものがありますか?」と聞いた
- 顧客が「片付けていて、おもちゃ・制作・私物の3タイプがあることがわかった」と自分で言語化
- 「職場では分かれているから片付くけど、家では全部一緒だから片付かない」と原因に気づいた
- この気づきが「仕組みが必要」という講座の価値に自然に繋がった

ポイント: 担当者が答えを言わず、質問で導く。

### パターン4: 将来の健康不安に紐づける

40代以降の顧客には、「もしもの時」の話が刺さりやすい。

事例（古賀いづみさま）:
- 顧客が「パートナーがいなかったら、甥っ子姪っ子に迷惑がかかる」と言った
- 担当者が「急な病気で倒れた時、お薬手帳を探してきてって言われたら?」と具体化
- 顧客は「今はちまちま言ってられない。緊急課題だ」と自分で結論を出した

ポイント: 「片付け」を「人生の緊急課題」に格上げするトーク。

### パターン5: ChatGPTで分割シミュレーションを見せる

お金の不安がある顧客には、分割払いの具体的なシミュレーションを「その場で」見せると効果的。

事例（小林友里さま 2026年4月）:
- 顧客が「やりたい気持ちは80点、残りはお金のところ」と言った
- 担当者がChatGPTで「66万円を楽天カード36分割、半年後に繰り上げ返済した場合」をシミュレーション
- 24分割と36分割の手数料差、繰り上げ返済後の総額を画面共有で見せた
- 顧客は「そんなに差がないんですね」と納得し、最終的に「現金一括の方が楽」と自分で結論を出した

ポイント: 数字を見せることで「高い」という感情的な反応を「払えるかどうか」という論理的な判断に変換する。

### パターン6: 「とりあえず」→「ついでに」で成果を体感させる

言葉の変化を示すことで、講座の効果を具体的に伝える。

事例（小林友里さま 2026年4月）:
- 担当者が「私は講座に入る前、『とりあえず』ばかり言っていた」と自己開示
- 「4〜5ヶ月目に、自分が『とりあえず』と言わなくなっていることに気づいた」
- 「代わりに『ついでに』と言っていた。立ち上がった時に『ついでにこれ片付けとこう』」
- 顧客が「ああ、なるほど」と共感

ポイント: 抽象的な「変わりました」ではなく、具体的な言葉の変化を示すと伝わりやすい。

### パターン7: 不登校の子供がいる顧客への共感

子供の不登校を抱える顧客には、「家の状態と子供の状態は関係がある」というメッセージが刺さる。

事例（小林友里さま 2026年4月）:
- 顧客が「長女が不登校から復帰した」「娘が頑張っているから私も変わりたい」と語った
- 担当者が「1秒収納の講座生には不登校の子供を持つ方も多い」と共感
- 「家が休まらないのも1つの背景」「片付けたら学校に行くようになった子もいる」と事例を紹介
- 「引きこもりの息子が、ソファを買いに行く時に出てきて手伝ってくれた」という涙の事例

ポイント: 「片付け」を「子供のため」という文脈に置き換えると、顧客の動機が強化される。

### パターン8: ナチュラルウーマンクラブの画面を見せる

講座の内容を説明するだけでなく、実際のコミュニティ画面を見せることで具体性を高める。

事例（小林友里さま 2026年4月）:
- 担当者がDiscordの画面を共有し、スケジュール、勉強会一覧、作業会呼びかけの様子を見せた
- 「こんなに充実している講座ってなかなかない」と価値を視覚化
- 顧客が「楽しそうですね」「すごいですね」と反応

ポイント: 言葉で説明するより、実物を見せた方が伝わる。

### パターン9: 顧客が泣いた時の受け止め方

顧客が泣いた時は、否定せず、そのまま受け止める。泣くほど心を開いている証拠。

事例（市原千鶴さま 2026年4月）:
- 講師が「真面目で頑張り屋さんだからこそ自分を追い込んでしまう」と伝えた
- 顧客が泣いた（開始5分で涙）
- 講師の対応:
  - 「全然いいですよ」
  - 「涙って結構ストレス発散になりますからね」
  - 「まだ始まって何分も経ってないですけど、全然大丈夫ですよ」
  - 「それだけやっぱり張り詰めてこられたんですよ」
  - 「辛かったですね」
- 顧客はその後、夫との関係、転職の不安、無駄遣いの悩みなど深い本音を語った

ポイント:
- 「泣かないでください」「大丈夫ですか?」と言わない
- 泣くことを肯定する（ストレス発散、自然な反応）
- 共感の言葉を添える（「辛かったですね」「張り詰めてたんですね」）

### パターン10: 顧客の言葉を繰り返して承認する

顧客が語った内容を、ポジティブな言葉で言い換えて返す。

事例（市原千鶴さま 2026年4月）:
- 顧客が「娘のこと、仕事のこと、家のこと、色々背負っている」と語った
- 講師が「守る人ですね、千鶴さん。素晴らしいですよ」と返した
- 顧客が「管理職だったけど年収下げて転職した」と語った
- 講師が「この若さで管理職すごい」「頭がいいから先回りして分かっちゃう」と返した
- 顧客が「課題は分かってるけど動けない」と語った
- 講師が「課題をちゃんと見つめるのが上手でいらっしゃいますね」と返した

ポイント:
- 顧客が自分を否定する言葉を使っても、ポジティブに言い換える
- 「〜ですね」と断定形で返す（「〜かもしれませんね」より強い）
- 名前を添える（「千鶴さん」）

### パターン11: 反面教師を活用した緊急性の引き出し

顧客が語った身近な「片付けられない人」のエピソードを、緊急性を引き出すために活用する。

事例（山中明子さま 2026年4月）:
- 顧客が「90歳のおばのマンションがとんでもないことになっている」と語った
- 「電気が切れている」「風通しが悪い」「ケアマネが入れない」という深刻な状況
- 担当者が「そこまで行っちゃったら取り返しつかないですね」と共感

## 失敗パターン集

### パターン1: 環境確認の不足

### パターン2: 価値の棚卸しをせずに価格提示

対策: 価格提示前に「棚卸しトーク」で顧客自身に価値を語らせる。

### パターン3: クロージングで引きすぎる

対策: 「背中を押させてください」のスタンスで、決断を促す。

### パターン4: 「やりたい気持ち50%」の深掘り不足

顧客が「やりたいけど半々」と言った時、残りの50%が何かを聞かないまま進むと、クロージングで詰まる。

事例（古賀いづみさま）:
- 顧客が「やりたい気持ちは100点中50点」と回答
- 担当者は「どうしてやりたいと思いますか?」と「やりたい側」だけを聞いた
- 「やりたくない側」「迷っている側」の理由を深掘りしなかった
- 結果、お金の不安が解消されないまま進んだ

対策: 「残りの50%、迷っている部分って何ですか? お金? 時間? それとも『本当に私でもできるか』っていう不安?」と具体的に聞く。

### パターン5: 分割プランの具体性不足

「分割もできますよ」だけでは、顧客のお金の不安は解消されない。

対策: 「頭金〇万円、月々〇万円の12回払いなら、月〇万円です。今の生活で〇万円なら捻出できそうですか?」と具体的な数字を出す。

### パターン6: 高熱意なのに「やりましょうよ！」を言わず保留

顧客が「80〜90点」の熱意を示しているのに、「やりましょうよ！」と背中を押さないと保留になる。

事例（杉浦彩佳さま 2026年4月）:
- 顧客が「やりたい気持ちは80〜90点」と言った

- 顧客が「即頭できない」「金額がかかる」と言った時、「そうですね」で終了
- 結果、保留
- 80点以上の熱意があれば、背中を押すだけで成約できた可能性が高い

対策:

- 残りの20〜30%が何かを聞き、その場で解消する
- 引きすぎない。「私からは進めません」ではなく「背中を押させてください」のスタンスで

### パターン7: 成約後キャンセル防止トークの不実施 → キャンセル発生
